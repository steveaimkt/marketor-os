③커맨드 15종 · 여러 카드가 이어달리는 묶음 업무

업무 카드가 명사(1장짜리 업무)라면 커맨드는 동사("~돌려줘")입니다. 한 마디에 카드 3~4장이 순서대로 이어달리고, 앞 카드의 결과물이 다음 카드에 자동으로 입력됩니다. 당신은 단계 사이 체크포인트(⏸)에서만 답하면 됩니다. 업무 생애주기 5단계로 묶여 있습니다.

W1 진단·리서치일을 시작하기 전, 시장·고객을 알아본다

  • "시장리서치 돌려줘"키워드 리서치 → 경쟁사 분석 → 리뷰 마이닝 → 가격 조사
  • "VOC애널리틱스 돌려줘"리뷰 마이닝 → VOC 분류 → 개선 우선순위 → 리뷰 관리

W2 구축·런칭새로 세운다: 브랜드 · 제품 · 스토어 · OS

  • "브랜드셋업 돌려줘"브랜드 스토리 → 가이드라인 → 페르소나 → 가치제안
  • "신제품런칭 돌려줘"컨셉 브레인스토밍 → 가치제안 → 가격 전략 → 런칭 플랜
  • "스토어셋업 돌려줘"상품명 최적화 → 상세페이지 → Figma 디자인 → 정책 설계
  • "마케터OS셋업 돌려줘"자동화 진단 → 나만의 스킬 만들기 → 오케스트레이션

W3 제작·캠페인알린다: 콘텐츠 · 채널 · 협업 운영

  • "콘텐츠프로덕션 돌려줘"캘린더 → 블로그(또는 유튜브 대본) → 멀티채널 변환 → 성과 회고
  • "유튜브프로덕션 돌려줘"트렌드 스캔 → 스크립트 → 썸네일·제목 A/B → 쇼츠 변환
  • "소셜그로스 돌려줘"채널 진단 → 피드 기획 → 바이럴 훅 → 이벤트 설계
  • "인플루언서캠페인 돌려줘"발굴·평가 → 계약·브리프 → UGC 활용 → 멀티채널 변환

W4 분석·최적화잰다: 성과를 측정하고 조정한다

  • "광고애널리틱스 돌려줘"주간 리포트 → 쿠팡(네이버) 분석 → 예산 재배분 → 소재 카피
  • "월간리포트 돌려줘"데이터 클리닝 → 판매 분석 → 월간 보고서 → KPI 트리

W5 관계·대응지킨다: 고객 관계 · 수주 · 위기

  • "리텐션캠페인 돌려줘"RFM 세그먼트 → 고객 여정 → 이메일 시퀀스 → 윈백 캠페인
  • "세일즈파이프라인 돌려줘"미팅 노트 → 제안서 → 세일즈 덱 → 제휴 제안
  • "위기커뮤니케이션 돌려줘"소셜 리스닝 → 대응 플랜 → 응대 (+보도자료)

커맨드의 속커맨드가 정하는 것은 딱 3가지

  • ①카드 호출 순서 (체인)
  • ②단계 사이 체크포인트 ⏸ (사용자 게이트)
  • ③끝난 뒤 다음 커맨드 자연어 제안

④요청 1건의 여정 · "유튜브 콘텐츠 분석해줘"가 처리되는 길

모든 요청이 같은 길을 갑니다. 멈춤 표시(⏸ 결재)가 있는 곳에서만 당신의 답을 기다립니다. 나머지는 알아서 굴러갑니다.
