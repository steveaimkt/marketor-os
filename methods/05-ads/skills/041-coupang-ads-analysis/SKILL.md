# 041 쿠팡 광고 성과 분석

> 쿠팡 광고센터 키워드 리포트 1개 → KPI 분해 → 효자/낭비/잠재 분류 → 입찰 조정안 + 긴급·주간·월간 실행 계획.
> 계승: 안티그래비티 「쿠팡 광고 분석 에이전트」 — KPI 공식 5종 · 키워드 분류 체계 · ROAS 원인 트리 · 지면 분석 로직.

## Contract
- 판정 기준은 [brand/profile.md](../../../../brand/profile.md) §4 **손익분기 ROAS(=1/마진율)** 다. 절대값 하드코딩 금지 — profile 미작성 시에만 원본 기본값(BEP 100%·목표 300%)을 쓰고 `[추정]` 태그를 붙인다.
- 전환 지표는 **14일 전환 기준**으로 통일하고 산출물에 기준을 명시한다 (1일 전환과 혼용 금지).
- 낭비/효자 판정마다 근거 수치(광고비·주문수·ROAS)를 병기한다 — 근거 없는 키워드 지목 금지.
- 입찰 조정안은 제안까지만. 실제 키워드 OFF·입찰 변경은 사용자가 광고센터에서 수동 실행한다.
- 같은 리포트를 다시 넣으면 같은 분류가 나온다 (분류 임계값을 산출물 ①에 명시 — 재현성).

## Phases
1. **기준 로드** — profile.md §4에서 손익분기 ROAS·목표 ROAS·AOV 자동 참조 (재질문 금지).
2. **데이터 확보** — [data/connections.md](../../../../data/connections.md) 기준 쿠팡은 공식 MCP 없음 → **광고센터 키워드 리포트 파일 업로드가 기본 경로** (광고센터 → 리포트 → 키워드 리포트 → 기간 설정 → 다운로드). ⚠️ 캠페인/광고그룹 리포트는 컬럼 구조가 달라 호환 불가 — 형식 검증 후 진행. 실데이터 `[실데이터]` / 샘플 `[샘플]` 태그.
3. **KPI 계산** — CTR=클릭/노출 · CPC=광고비/클릭 · CVR=주문/클릭 · ROAS=매출/광고비 · CPA=광고비/주문. 캠페인×지표 매트릭스 + 일별 추이.
4. **키워드 분류** — BEP 기준으로 상대화한 분류 체계 (원본 임계값은 profile 미작성 시 폴백):

| 분류 | 조건 | 액션 |
|---|---|---|
| 효자 (Star) | ROAS ≥ 목표 ROAS + 주문 있음 | 수동 캠페인 등록·입찰 상향 |
| 양호 | BEP ≤ ROAS < 목표 + 주문 있음 | 현행 유지 |
| 잠재 | 주문 있으나 ROAS < BEP | 입찰 하향·상세페이지 개선 |
| 낭비 (Waste) | 광고비 ≥ 5만원 + 주문 0건 | 즉시 OFF 또는 제외 키워드 |
| 관찰 | 광고비 있음 + 주문 0건 (임계 미달) | 관찰 후 판단 |

5. **지면 분석** — 검색/비검색/리타겟팅 지면별 ROAS·CPC 비교 → 지면 입찰 가중치 제안.
6. **원인 트리 진단** — ROAS = CTR × CVR × AOV ÷ CPC 로 분해, 각 요소 상/중/하 판정 → 병목 우선순위 CTR→CVR→CPC→AOV.
7. ⏸ **실행 계획 확정** — 긴급(즉시 OFF 낭비 키워드) / 주간(입찰 조정) / 월간(캠페인 재편) 3단 계획을 제시하고 사용자 승인 후 마무리 → 체인: 046 예산 재배분(채널 차원) · 043 소재 카피 교체(CTR 병목 시).

## Anti-Patterns
- **1일 전환으로 판정** — 쿠팡 리포트의 1일/14일 전환을 섞으면 ROAS가 절반으로 보인다. 14일 기준 고정.
- **합산 ROAS만 보고 종료** — 캠페인 합산이 흑자여도 키워드 단위 낭비는 숨는다. 반드시 키워드 레벨까지 분해.
- **업종 불문 300% 기준** — 마진 20% 브랜드의 BEP는 500%다. 기준은 항상 profile.md 손익분기 ROAS에서.
- **관찰 키워드 조기 처형** — 광고비 소액+주문 0을 바로 OFF하면 신규 키워드 학습 기회를 죽인다. 낭비 임계 미달은 관찰로.
- **분석만 하고 실행 경로 생략** — 조치별 광고센터 실행 위치(키워드 관리·수동 캠페인·지면 설정)까지 안내해야 리포트가 실행된다.

## 활용
- **주간 루틴**: 매주 월요일 지난주 키워드 리포트 다운로드 → 본 스킬 실행 → 낭비 OFF·효자 수동 등록 → 046으로 주간 예산 재배분까지 한 흐름.
- **신제품 초기 진단**: 집행 2주차 리포트로 원인 트리만 먼저 — CTR 병목이면 043 소재, CVR 병목이면 051 상세페이지로 분기.
- **월간 캠페인 재편**: 월말 4주치 리포트로 캠페인 처방(확장/유지/축소/개편/상품개선)을 받아 다음 달 구조 확정.
