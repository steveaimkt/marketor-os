# 044 구글·유튜브 광고 세팅 가이드

> 목표·예산 하나로 검색/PMax/유튜브 캠페인 구조 설계안 + 타깃팅·전환 추적 체크리스트 + 품질점수 사전 점검표.
> 계승: 패스트캠퍼스 6-2 google-ads-analyzer — 품질점수 구성(예상 CTR·광고 관련성·LP 경험→Ad Rank→CPC) · 키워드 vs 검색어 · 브랜드 방어 로직을 **사후 분석에서 사전 세팅으로 방향 전환**.

## Contract
- 캠페인 구조는 목표(전환/트래픽/조회·인지) × 월 예산 매트릭스의 표준 패턴을 따르고, **예산 규모에 맞는 최소 구조로 시작**한다 — 캠페인·광고그룹 수마다 근거를 병기하며 과잉 세분화(SKAG류)를 금지한다.
- 브랜드 키워드 캠페인은 일반 키워드와 **항상 분리**한다 (방어 예산 확보 + 합산 ROAS 착시 방지).
- 전환 추적(GA4 연동·향상된 전환) 완료가 체크리스트 1순위 — 미완료 상태의 PMax 개설은 보류로 판정한다.
- 타깃 ROAS 입찰 목표값은 [brand/profile.md](../../../../brand/profile.md) §4 **손익분기 ROAS** 기준으로 산출한다 — 미작성 시 폴백값 사용 + `[추정]` 태그.
- 이 스킬은 설계안까지만 — 실제 계정 세팅·집행 개시는 사용자가 Google Ads에서 수동 실행한다.

## Phases
1. **기준 로드** — profile.md §4에서 손익분기 ROAS·목표 ROAS·월 마케팅 예산 자동 참조 (재질문 금지).
2. **목표·예산 확정** — 주 목표 1개 선택(전환/트래픽/조회·인지). 복수 목표 요청 시 캠페인을 목표별로 나눈다는 원칙 고지.
3. **캠페인 구조 설계** — 예산 단계별 표준 패턴:

| 월 예산 | 구조 | 근거 |
|---|---|---|
| ~100만원 | 검색 2개 (브랜드 방어 + 일반 키워드) | 전환 데이터 축적이 최우선 — PMax·유튜브 보류 |
| 100~500만원 | 검색 2개 + PMax 1개 | 검색 전환 월 30건+ 확보 후 PMax 학습 가능 |
| 500만원~ | 검색 2~3개 + PMax + 유튜브(인스트림) | 유튜브는 인지 목표 분리 캠페인으로만 |

   광고그룹은 키워드 테마(구매 의도) 단위 3~7개 키워드 — 1키워드 1그룹 세분화 금지.
4. **매치타입·제외 키워드 사전 리스트** — 구문검색 중심으로 시작 → 런칭 후 검색어 리포트로 확장/제외 (키워드=내가 산 단어, 검색어=고객이 친 문장 — 그 틈에서 돈이 샌다). 정보성·무관 검색어(예: "유통기한", "영어로", 경쟁 시술명)를 제외 키워드로 **런칭 전에** 등록한다. 광범위 단일어(예: "토너") 입찰 금지.
5. **품질점수 사전 조건 점검** — 품질점수 3요소를 세팅 단계에서 충족:

| 요소 | 사전 조건 | 미충족 리스크 |
|---|---|---|
| 예상 CTR | 광고문구에 키워드 포함 + 반응형 검색광고 제목 10개+ | 노출 순위 하락 |
| 광고 관련성 | 광고그룹 키워드-광고문구 테마 일치 | 같은 노출에 CPC 상승 |
| LP 경험 | 키워드-광고-랜딩 3자 정합 + 모바일 속도 | 점수 5~6점 → CPC 할증 |

6. **타깃팅·전환 추적 체크리스트** — 위치(대한민국·"현재 위치" 설정), 언어(한국어+영어 병기 주의), 잠재고객 세그먼트(관찰 모드로 시작), 기기·광고 일정, 콘텐츠 제외(앱 지면), GA4 연동·향상된 전환·리마케팅 태그. 각 항목 ☐ 체크박스로 제공.
7. ⏸ **세팅안 확정** — 설계안 승인 후 마무리 → 체인: 집행 2주 후 045 광고 주간 리포트로 성과 점검 루프 시작.

## Output Format
```
## 구글·유튜브 광고 세팅안 — {브랜드} · 목표: {전환|트래픽|인지}
판정 요약: 월 예산 {n}원 → {검색 n + PMax n + 유튜브 n} 구조 · 목표 tROAS {n} (profile.md §4 BEP {n} 기준)

### ① 캠페인 구조도 (캠페인 → 광고그룹 → 키워드 테마 · 각 예산 배분 + 근거)
### ② 키워드 시드 & 제외 키워드 사전 리스트 (매치타입 병기)
### ③ 품질점수 사전 점검표 (3요소 × 충족 여부 × 조치)
### ④ 타깃팅·전환 추적 체크리스트 ☐ (1순위: GA4 연동·향상된 전환)
### ⑤ 런칭 순서 ⏸ (전환 추적 → 검색 → [전환 30건+/월 후] PMax → 유튜브)

다음 액션: → 집행 2주 후 045 광고 주간 리포트 자동화 (검색어 리포트로 제외 키워드 갱신)
```

## Anti-Patterns
- **첫날부터 PMax** — 전환 데이터 없는 PMax는 학습 실패로 예산만 태운다. 검색 전환 월 30건+ 확보가 선행 조건.
- **브랜드·일반 합산 설계** — 브랜드 키워드의 높은 ROAS가 일반 키워드 적자를 가린다. 캠페인 분리가 세팅의 1번 규칙.
- **광범위 단일어 입찰** — "토너" 같은 단일어는 CPC만 높고 의도가 안 잡힌다 (6-2 실측: 품질점수 5점·CPC 760원 낭비 사례).
- **전환 추적 없이 집행 개시** — 추적 없는 2주는 데이터가 아니라 소음이다. 체크리스트 1순위를 건너뛰면 세팅 미완료.
- **키워드 1개 = 캠페인 1개 세분화** — 관리 불능 + 학습 데이터 분산. 예산 규모에 맞는 최소 구조가 원칙.

## 활용
- **첫 집행 셋업**: 신규 브랜드가 월 100만원으로 시작 — 검색 2캠페인 최소 구조 + 제외 키워드 사전 리스트로 초반 누수 차단.
- **PMax 전환 판단**: 검색 캠페인 성과표를 들고 "PMax 열어도 되나"를 구조 기준(전환 30건+/월)으로 판정.
- **품질점수 예방 정비**: 집행 중 CPC가 오르면 ③ 사전 점검표를 역으로 적용 — 점수 7 미만 키워드의 LP 정합부터 보수 (048 CRO 진단과 세트).
