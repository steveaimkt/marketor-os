# 049 광고 정책·심의 체커

> 집행 전에 잡는다. 매체 정책 × 업종 규제 이중 스캔 → 차단/주의/통과 3단 판정 → 소구력을 보존하는 대체 표현까지.
> 계승: gates/compliance-gate.md 3 STEP 방법론(룰 로드→전수 스캔→판정 리포트) + 패스트캠퍼스 5-3 규제 치환(금기어 검출→자동 치환→치환 로그, 사후 검열이 아닌 사전 차단).

## Contract
- 매체 정책과 업종 규제([brand/compliance.md](../../../../brand/compliance.md) — 표시광고법 공통 + 업종 섹션)를 **모두** 검사한다. 한쪽만 통과한 소재를 "통과"로 판정하지 않는다.
- 판정은 ⛔ 차단(치환 불가한 핵심 주장 위반) / ⚠️ 주의(치환 가능·근거 필요) / ✅ 통과 3단계, 문장 단위 검출 내역을 첨부한다.
- 수정 제안은 소구력을 최대한 보존하는 대체 표현으로 낸다 — 밋밋하게 삭제만 하는 제안 금지 (치환 사전 우선: 재생→활력, 치료→케어, 항염→진정).
- 업종은 [brand/profile.md](../../../../brand/profile.md) §1의 규제 카테고리 값에서 자동 로드한다 (재질문 금지, 미작성 시 사용자 확인 1회).
- 이 스킬은 사전 검토 참고용 — 최종 심의 판단은 각 매체·기관에 있음을 산출물에 고지한다.

## Phases
1. **컨텍스트 로드** — profile.md §1 업종 + §6 금기·필수 고지 + compliance.md 해당 업종 섹션 로드.
2. **매체 정책 검사** — 게재 매체별 심의 기준 차이를 적용해 스캔:

| 항목 | 메타 | 네이버 | 쿠팡 | 구글 |
|---|---|---|---|---|
| 심의 방식 | 사후 자동+수동 (거부 시 계정 품질 하락) | **사전 검수** (영업일 1~2일, 반려 사유 통보) | 입점사 소재 사전 검수 | 사후 자동 (반복 위반 시 계정 정지) |
| 민감 요소 | 개인 특정("당신의 ○○"), Before/After 이미지, 신체 부위 클로즈업 | 최상급·보장 표현, 공식 문서 미비, 의료·건기식 사전심의필 | 타 플랫폼 언급·최저가 표현, 상표권 | 클릭베이트, 성인·건강 민감 카테고리, 상표 키워드 |
| 특이점 | 20% 텍스트 규칙은 폐지됐지만 텍스트 과다 시 도달 저하 | 업종 서류(기능성 인증 등) 첨부 필수 | 상품 상세와 소재 불일치 반려 | 랜딩페이지 정책 위반도 소재 거부 사유 |

3. **업종 규제 검사** — compliance-gate 3 STEP 준용: 금지 표현 검출 → 치환 사전 있으면 자동 치환, 없으면 대안 제안 → 필수 고지 누락(건기식 마크·대가성 표기) + 최상급·보장성 표현의 근거 유무 확인.
4. **판정·치환 로그** — 문장별 (원문 → 검출 사유 → 대체 표현) 로그 작성. ⛔ 검출 시 대체 불가 사유와 수정 방향 제시, 2회 연속 ⛔면 사용자 에스컬레이션.
5. **체인 제안** — 표시광고법 외 법무 쟁점(개인정보·경품 고지 등) 발견 시 → 099 법무 기초 체커. 성과 판정이 필요한 소재는 041/042/045/046 계열로 (그쪽 판정 기준 = profile.md 손익분기 ROAS).

## Output Format
```

판정: ✅ 통과 | ⚠️ 수정 후 통과 ({n}건 치환) | ⛔ 차단 ({사유})

### ① 매체 정책 검사 ({매체} 기준 {n}건)
- "{원문}" → {위험 요소} → 제안: "{대체 표현}"
### ② 업종 규제 검사 (compliance.md {업종} 섹션 {n}건)
- "{원문}" → {법령·조항} → 치환: "{대체 표현}"
### ③ 근거 필요 항목 — "{최상급 표현}" : 근거 자료 확보 전 사용 보류
### ④ 필수 고지 체크 — {건기식 마크|대가성 표기|...} {✅/누락}

※ 본 결과는 사전 검토 참고용이며 최종 판단은 각 매체 심의 기준을 따릅니다.
다음 액션: → 099 법무 기초 체커 (표시광고법 외 법무 쟁점 정밀 점검)
```

## Anti-Patterns
- **매체 정책만 보고 통과 선언** — 메타 자동 심의를 통과해도 화장품법 위반이면 행정처분 대상. 이중 스캔 필수.
- **검출만 하고 대안 없음** — "쓰지 마세요"로 끝나면 카피가 죽는다. 소구력 보존 대체 표현까지가 산출물.
- **전 매체 동일 기준 적용** — 네이버는 사전 검수·서류 요구, 메타는 사후+계정 품질. 매체별 표를 실제로 적용.
- **근거 있는 표현까지 일괄 삭제** — "판매 1위"는 근거 자료가 있으면 가능. 삭제가 아니라 근거 확보 요청으로.
- **통과 판정을 법적 보증처럼 서술** — 참고용 고지 누락 금지. 최종 책임 소재를 흐리면 안 된다.

## 활용
- **집행 전 최종 관문**: 043 카피 팩·051 상세페이지 산출물을 집행 직전 본 스킬로 매체 맞춤 재검수 — 반려로 인한 집행 지연(네이버 1~2일×재심의) 예방.
- **반려 원인 역추적**: 매체에서 반려된 소재를 넣으면 위반 후보와 수정본을 받아 재심의 1회로 통과.
- **다매체 동시 집행 준비**: 같은 소재를 메타·네이버·쿠팡·구글 4매체 기준으로 한 번에 스캔 — 매체별 수정본 분기.
- **팀 온보딩 교육**: 매체별 기준 차이 표를 신입 마케터 교육 자료로 — "왜 네이버만 서류를 내야 하나"를 표 한 장으로 설명.

> 참고 — 검사 우선순위: ⛔ 후보(효능·치료·보장 표현) → 근거 필요(최상급·1위) → 고지 누락(대가성·건기식 마크) → 매체 민감 요소 순. 심각한 위반부터 보고해 수정 동선을 줄인다.
> 게재 매체가 복수면 매체별로 본 스킬을 반복 실행하지 말고 한 번에 4매체 열을 나란히 출력한다 (동일 소재 1스캔 원칙).
