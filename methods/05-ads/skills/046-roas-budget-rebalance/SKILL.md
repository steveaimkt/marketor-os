# 046 ROAS 진단 & 예산 재배분

> 채널별 성과를 동일 잣대로 놓고 손익분기 ROAS 기준으로 판정 → 현재 배분 vs 효율 기준 배분 → 보수/중립/공격 3안 시나리오.
> 계승: 패스트캠퍼스 6-4 3media-integrated — 매체 비교표(동일 잣대) · 인사이트 5줄 · 현재 vs 제안 배분 + 근거 + 예상 효과 · "제안과 실행의 분리" 원칙.

## Contract
- **판정 기준 = [brand/profile.md](../../../../brand/profile.md) §4 손익분기 ROAS** (=1/마진율). 채널별 흑자/적자 판정은 전부 이 값 기준 — 미작성 시 BEP 1.0 폴백 + `[추정]` 태그.
- 재배분 시나리오는 보수/중립/공격 정확히 3안, 각 안에 예상 ROAS 변화·리스크·이동 금액을 수치로 병기한다.
- 매체 간 비교표에는 기여 모델 차이 주의 주석을 반드시 부착한다 (뷰스루 vs 클릭 기준).
- 재배분은 **제안까지만** — 실제 예산 집행 변경은 사용자 승인 영역, 실행 버튼은 없다.
- 같은 입력이면 같은 시나리오 수치가 나온다 (배분 산식과 가정을 산출물에 명시).

## Phases
1. **기준 로드** — profile.md §4 손익분기 ROAS·목표 ROAS·월 마케팅 예산 자동 참조 (재질문 금지).
2. **데이터 취합** — 입력 우선순위: ① 선행 041/042/045 산출물 (outputs/ 탐색) ② [data/connections.md](../../../../data/connections.md) 연결된 MCP 실조회 ③ **미연결 시 채널별 CSV 업로드 폴백**. 전 채널 동일 기간 정합성 검증.
3. **채널 기여 진단** — 채널별 지출·매출·ROAS·CPA 동일 잣대 비교표 + BEP 기준 신호등(🟢 BEP 초과 / 🟡 BEP 부근 ±20% / 🔴 BEP 미달). 예산 점유율 vs 매출 기여율 격차로 과소투자·과대투자 채널 검출.
4. **한계 효율 추정** — 채널별 최근 추세(지출 증가 구간의 ROAS 변화)로 증액 여력 판정 — "잘 되는 채널에 무한 증액" 방지.
5. **시나리오 3안 산출** —
   | 안 | 원칙 | 이동 폭 |
   |---|---|---|
   | 보수 | 🔴 채널 축소분만 🟢 채널로 | 총예산의 ~10% |
   | 중립 | 효율 비례 재배분 (기여율=점유율 수렴) | ~20% |
   | 공격 | 🟢 최상위 채널 집중 + 🔴 대폭 축소 | ~30%+ |
   각 안: 채널별 현재→제안 금액, 예상 통합 ROAS, 리스크(학습기 리셋·시즌 변수).
6. ⏸ **추천안 확정** — 1개 안을 근거와 함께 추천하고 사용자 선택을 받는다. 실행은 각 매체 관리자에서 수동.
7. **체인 제안** — 🔴 채널의 소재 병목이면 043 메타 광고 소재 카피 팩, 월 단위 예산 전체 설계는 095 마케팅 예산 플래너.

## Output Format
```

판정 요약: BEP ROAS {n} (profile.md §4) 기준 🟢 {채널} / 🔴 {채널} · 권장안: {보수|중립|공격} (예상 통합 ROAS {n}→{n})

### ① 채널 기여 진단표
| 채널 | 지출 | 매출 | ROAS | BEP 대비 | 예산 점유율 | 매출 기여율 | 판정 |
(⚠️ 매체 간 기여 모델 차이 — 직접 비교 시 주의)

### ② 시나리오 3안 비교표 (채널별 현재→제안 · 예상 ROAS · 리스크)
### ③ 추천안 + 근거 3줄 ⏸
### ④ 배분 산식·가정 (재현용)

다음 액션: → 043 소재 카피 팩 (🔴 채널 소재 교체) / 095 마케팅 예산 플래너 (월 예산 전체 재설계)
```

## Anti-Patterns
- **BEP 없이 ROAS 높낮이만 비교** — ROAS 2.0이 흑자인지는 마진이 결정한다. 판정은 반드시 profile.md §4 손익분기 ROAS로.
- **1안만 제시** — 리스크 성향은 사용자마다 다르다. 보수/중립/공격 3안 없는 재배분은 규격 위반.
- **효자 채널 무한 증액 가정** — 증액하면 한계 ROAS는 떨어진다. 한계 효율 추정 없는 공격안 금지.
- **매체 리포트 수치 그대로 합산** — 매체별 자가 기여 매출을 단순 합치면 중복 계상. 주의 주석 + 가능하면 실매출 대조.
- **제안을 실행처럼 서술** — "예산을 옮겼습니다" 금지. 이 스킬은 시나리오까지, 집행 변경은 사용자 몫.

## 활용
- **월말 예산 회의**: 045 주간 리포트 4장을 입력으로 월말 재배분 3안 생성 — "감으로 증액" 대신 비교표를 회의 자료로.
- **광고애널리틱스 체인**: /광고애널리틱스 커맨드(045→041/042→046→043)의 판정 허브 — 매체별 진단을 받아 예산 이동으로 수렴.
- **신규 채널 테스트 판정**: 테스트 채널의 2~4주 데이터로 BEP 대비 위치 확인 → 증액/철수 의사결정 근거.
- **사후 검증 루프**: 재배분 집행 2주 후 같은 형식으로 재실행 — 공격안의 한계 효율 가정이 맞았는지 실측으로 검증하고 다음 재배분의 가중치를 보정.

> 참고 — 시나리오 산식 기본형: 제안 배분비 = 현재 배분비 × (채널 ROAS ÷ BEP ROAS)^가중치, 가중치 보수 0.5 / 중립 1.0 / 공격 1.5. 실제 적용값·가정은 산출물 ④에 기록 (재현성).
