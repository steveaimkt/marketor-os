# 05-ads — 광고 운영 & 분석

## 이 카테고리가 해결하는 문제
광고비가 어디서 새는지 모른 채 집행하는 상태를 끝낸다. 쿠팡·네이버·메타·구글 매체별 진단과 주간 리포트로 성과를 상시 가시화하고, ROAS 진단으로 예산을 재배분한다. 소재 쪽에서는 카피 팩 생성 → A/B 판독 → 정책·심의 체크까지 집행 전후를 모두 커버한다.

## 추천 진입 경로
- **대표 커맨드: /광고애널리틱스** — 체인 `045 → 041(042) → 046 → 043` (주간 리포트 취합 → 쿠팡(또는 네이버) 매체 진단 → ROAS 진단·예산 재배분 → 다음 소재 카피 팩)
- **처음이면 045(광고 주간 리포트 자동화)부터** — 전 매체 성과가 한 장으로 모여야 어느 매체를 깊게 진단할지 정할 수 있다.

## 다른 카테고리와의 연결
- **→ 07-analytics**: 050(어트리뷰션 정리) → 063(GA4 분석 리포트) — UTM 체계가 서야 트래픽·전환 해석이 정확해진다
- **→ 10-ops**: 046(ROAS 진단 & 예산 재배분) → 095(마케팅 예산 플래너), 049(광고 정책·심의 체커) → 099(법무 기초 체커) — 재배분 시나리오와 규제 검토가 운영 레이어로 이어진다
