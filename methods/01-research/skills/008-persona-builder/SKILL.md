# 타깃 페르소나 빌더

> 고객 데이터에서 구매자 패턴 클러스터링 → 페르소나 카드 3~5종 + JTBD 1문장 → 대표 페르소나 구매 여정 → 채널·메시지 방향 연결.
> 계승: `/시장조사` 페르소나 단계의 두 결정 원칙 — "리뷰에서 나타나는 구매자 패턴에서 도출"(교과서 페르소나 방지) + "각 항목에 데이터 근거 표시"(허구 방지) — 과 인지→재구매 구매 여정 프레임. 독립 스킬로 분리하며 3~5종·JTBD 정의 보강.

## Contract
- 페르소나는 입력 데이터의 구매자 패턴에서만 도출된다 — 데이터에 없는 유형을 상상으로 추가하지 않으며, 데이터가 없으면 진행하지 않고 최소 입력(리뷰 50건 또는 인터뷰 3건 상당)을 요청한다.
- 페르소나 카드의 모든 항목에는 근거(리뷰 발췌·구매 패턴·설문 수치)가 붙는다. 데이터 없이 채운 항목은 [추정] 태그.
- 각 페르소나에는 JTBD 1문장이 고정 형식으로 붙는다: "{상황}일 때, {동기}하고 싶다. 그래서 {제품}을 고용한다."
- 페르소나는 서로 다른 니즈·행동 축으로 3~5종을 만든다 — 나이만 다른 복제 페르소나 금지.
- 각 페르소나는 마케팅 활용처(주력 채널 + 메시지 방향 1줄)까지 연결되어야 완성이다.

## Phases
1. **브랜드 컨텍스트 로드** — [brand/profile.md](../../../../brand/profile.md) 자동 참조 (제품·가격대·기존 타깃 가설). 브랜드 정보 재질문 금지.
2. **입력 데이터 검토** — 리뷰·구매 데이터·설문(007)·리뷰 마이닝 산출물(006)·인터뷰(080) 로드. 소스별 건수·품질 보고. **데이터 소스 미확보 시**: 진행 중단하고 필요 데이터 안내 — 일반론 페르소나 생성 금지 (부득이 시장 공개 정보만으로 진행 시 전체에 [추정] 태그 + "검증 필요" 명시).
3. **패턴 클러스터링** — 구매 동기·사용 맥락·불만 축으로 데이터를 군집화해 3~5개 구매자 패턴 도출. 각 패턴의 규모(해당 리뷰/응답 비중)를 기록 — 소수 패턴(5% 미만)은 페르소나 승격 보류.
4. **페르소나 카드 작성 (3~5종)** — 카드당: 이름·나이대·직업·라이프스타일 / 구매 동기 / 핵심 니즈 / 가장 큰 불만 / 정보 탐색 채널. **모든 항목 옆에 근거 표기** (예: "성분 확인 후 구매 — 리뷰 23건 패턴").
5. **JTBD 정의** — 페르소나별 1문장: "{상황}일 때, {동기}하고 싶다. 그래서 {제품}을 고용한다." 기능적 job 외에 감정적·사회적 차원이 데이터에 보이면 병기.
6. **대표 페르소나 구매 여정** — 규모 1위 페르소나의 인지→탐색→비교→구매→사용→재구매 여정: 단계별 채널·찾는 정보·감정·데이터 근거.
7. **채널·메시지 연결 + 체인 제안** — 페르소나별 주력 채널 + 메시지 방향 1줄 매핑. 012 USP·가치제안 정의 (JTBD → 가치제안문) · 043 메타 광고 소재 카피 팩 (페르소나별 오디언스 소재) 추천으로 마무리.

### 지원 소스 (입력으로 받을 수 있는 것)
- 리뷰 데이터 (006 리뷰 마이닝 산출물이 최상 — 5분류·클러스터 재활용)
- 구매 데이터 CSV (재구매 주기·객단가·옵션 선택 패턴)
- 설문 응답 (007 세그먼트 교차표) · 인터뷰 녹취·요약 (080)

## Anti-Patterns
- **교과서 페르소나** — "트렌드에 민감한 2030 여성" 같은 어디서나 통하는 인물은 아무 결정도 못 돕는다. 데이터 패턴에서만 도출한다.
- **근거 없는 디테일** — 서사를 위해 지어낸 취미·성격은 허구와 실제를 구분 불가능하게 만든다. 근거 표기 없는 항목 금지.
- **나이만 다른 복제** — 25세/35세/45세 버전 하나씩은 세그먼트가 아니다. 니즈·행동 축이 달라야 별도 페르소나다.
- **JTBD 없는 카드** — 인구통계 카드는 장식이다. "무엇을 해결하려 고용하는가"까지가 페르소나다.
- **활용 연결 생략** — 채널·메시지 방향 없이 끝내면 다음 스킬(012·043)이 재분석을 반복하게 된다.

## 활용
- **광고 오디언스 설계**: 페르소나 3종을 043에 넘겨 오디언스별 소재 세트를 분리 제작 — 단일 소재 대비 반응 축이 명확해진다.
- **상세페이지 타깃 정렬**: 대표 페르소나의 구매 여정 "비교" 단계에서 찾는 정보를 051 상세페이지 섹션 순서에 반영한다.
- **분기 갱신**: 신규 리뷰·설문이 쌓일 때마다 재실행해 페르소나 비중 변화를 추적 — 주력 세그먼트 이동을 조기 감지한다.
