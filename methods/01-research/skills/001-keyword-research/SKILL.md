# 키워드 수요 리서치

> 시드 키워드 1개 → 연관어 확장 → 검색량·경쟁도 점수화 → TOP 20 우선순위 + 콘텐츠 제안.
> 계승: marketing-os `seo-keyword-research`의 확장 → 점수화 공식 → TOP 20 → 콘텐츠 제안 로직.

## Contract
- 시드 키워드 1개만 받으면 연관 키워드 50개 이상을 확장·평가한다.
- 모든 키워드에 우선순위 점수 `검색량 정규화(0~100) × 전환가능성 / 경쟁도`가 산출 근거와 함께 붙는다.
- 브랜드 카테고리·페르소나와 맞지 않는 키워드는 -50% 보정을 명시하고 반영한다.
- 검색량·경쟁도 수치에는 출처를 표기하고, 조회 불가 시 "추정" 라벨을 붙인다.
- 산출물은 항상 TOP 20 표 + 콘텐츠 제안 5건 + 다음 스킬 추천으로 끝난다.

## Phases
1. **브랜드 컨텍스트 로드** — [brand/profile.md](../../../../brand/profile.md) 자동 참조 (업종·핵심 제품·타겟·운영 채널). 브랜드 정보 재질문 금지. 비어 있으면 `/마케터OS셋업` 안내 후 일반 모드로 진행.
2. **확장** — 시드 키워드 → 연관어 50~100개 (자동완성 + 연관 검색어 + LSI 변형).
3. **정량 분석** — 키워드별: 월간 검색량(네이버 + 구글) · 경쟁도(CPC·광고주 수) · SERP 유형(블로그/영상/쇼핑/이미지).
   - **데이터 소스**: 네이버 데이터랩·검색광고 키워드도구 연동 시 실측 / 미연결 시 SERP 관찰 기반 추정 모드가 기본 (모든 수치에 [추정] 표기). 연동 방법은 [data/connections.md](../../../../data/connections.md) 참조.
4. **점수화** — `priority = 검색량 정규화(0~100) × 전환가능성 / 경쟁도`. 브랜드 적합성 미달 시 -50% 보정, 계절성 키워드는 피크 월 표기.

   **등급 → 수치 변환표 (점수 공식용)**
   | 등급 | 전환가능성 | 경쟁도 |
   |---|---|---|
   | 높음 | 3 | 3 |
   | 중간 | 2 | 2 |
   | 낮음 | 1 | 1 |

   검색량 정규화: 후보군 내 최대 검색량 = 100 기준 선형 환산 (점수 범위 0~300).
   예: 정규화 100 · 전환 높음(3) · 경쟁 낮음(1) → 100 × 3 ÷ 1 = **300** / 정규화 50 · 높음(3) · 낮음(1) → **150**.

5. **TOP 20 추출 + 콘텐츠 제안** — 상위 5개 키워드 각각에 포맷(블로그/영상/SNS)과 앵글 1줄 제안.
6. **산출 + 체인 제안** — 002 경쟁사 분석 리포트 · 022 블로그 SEO 아티클 추천으로 마무리 (보조: 021 콘텐츠 캘린더 빌더).

### 등급 기준 (점수화에 사용)
| 항목 | 높음 | 중간 | 낮음 |
|---|---|---|---|
| 전환 가능성 | 구매·비교 의도어 포함 ("추천", "가격", "후기") | 정보 탐색어 | 단순 호기심어 |
| 경쟁도 | CPC 상위 25% 또는 광고주 다수 | 중간 | 광고 거의 없음 |
| 브랜드 적합성 | profile.md 카테고리·페르소나 일치 | 인접 카테고리 | 무관 (-50% 보정) |

|---|---|---|---|---|---|---|---|

## Anti-Patterns
- **검색량 큰 키워드만 상위 배치** — 경쟁도 낮은 롱테일이 ROI가 높다. 반드시 공식대로 정렬한다.
- **근거 없는 수치 생성** — 조회 실패 시 그럴듯한 숫자를 지어내지 말 것. "추정" 표기 또는 제외.
- **브랜드 무관 키워드 나열** — profile.md 카테고리 밖 키워드를 보정 없이 올리지 않는다.
- **확장 생략** — 시드 유사어 10개로 끝내면 미달. 50개 미만 확장은 재실행.
- **계절성 무시** — 검색량이 특정 월에 몰린 키워드를 연중 키워드처럼 추천하지 않는다.

## 활용
- **신제품 출시 전**: 제품명 + 고민 키워드로 돌려 상세페이지·블로그의 타깃 검색어를 확정한다.
- **월초 콘텐츠 기획**: TOP 20을 021 콘텐츠 캘린더의 입력으로 넘겨 주간 30건 소재로 전개한다.
- **광고 키워드 그룹 설계**: 점수 상위 키워드를 검색광고 키워드 그룹 초안으로 재활용한다.
