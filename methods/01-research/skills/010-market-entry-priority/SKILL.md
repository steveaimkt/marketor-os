# 진입 시장 우선순위

> 진입 후보 2~7개 → Impact(시장 크기×적합도) × Effort(진입 비용+경쟁 강도) 채점 → 4분면 매트릭스 → TOP 3 + 1순위 90일 액션.
> 신규 설계 — 091 자동화 진단의 2축 매트릭스 문법(점수화→4분면→★지금 바로→근거 병기)과 동형. 004 SOM 산출물을 Impact 입력으로 연동.

## Contract
- 평가 기준(Impact·Effort 세부 항목과 가중치)을 채점 전에 먼저 제시하고 사용자 합의를 받는다 — 채점 후 기준 변경 금지.
- 모든 점수에는 근거 데이터 또는 "가정: {근거}"가 병기된다. 데이터 없는 항목은 [추정] 태그.
- 모든 후보는 같은 기준·같은 척도(각 항목 1~5)로 채점되어 4분면에 좌표를 갖는다 — 후보별로 다른 잣대 금지.
- TOP 3는 점수순 나열이 아니라 사분면 위치 + 자원 제약 반영 판정이며, 1순위에는 90일 액션(30/60/90일 3단)이 붙는다.
- 004 시장 규모 산출물이 있으면 SOM 보수값을 시장 크기 점수의 근거로 우선 사용하고 출처를 표기한다.

## Phases
1. **브랜드 컨텍스트 로드** — [brand/profile.md](../../../../brand/profile.md) 자동 참조 (핵심 역량·기존 채널·팀·예산 규모). 브랜드 정보 재질문 금지.
2. ⏸ **후보·기준 확정** — 후보 2~7개 목록 + 평가 기준 확정을 사용자에게 합의받는다:
   - **Impact = 시장 크기 × 적합도** — 시장 크기(SOM/검색량/카테고리 매출 규모) · 적합도(기존 역량·브랜드·고객과의 시너지)
   - **Effort = 진입 비용 + 경쟁 강도** — 진입 비용(초기 투자·신규 학습·운영 부담) · 경쟁 강도(상위 사업자 집중도·차별화 난이도)
   - 기본 가중치는 균등, 사용자 요청 시 조정 (조정 내역 산출물에 기록).
3. **후보별 채점** — 4개 항목을 각 1~5점, 근거 병기. **데이터 소스**: 004 산출물·001 검색량·002 경쟁 지형 산출물 우선 / 없으면 공개 정보 조사 / 그마저 없으면 가정 명시 + [추정] 태그. `Impact = 시장크기 × 적합도` (1~25) / `Effort = 진입비용 + 경쟁강도` (2~10).
4. **4분면 매트릭스 배치** — Impact(세로) × Effort(가로), 경계선은 후보군 중앙값:
   - ★ **지금 진입** (고Impact·저Effort) / **다음 라운드** (고Impact·고Effort — 자원 확보 후) / **자투리** (저Impact·저Effort — 부수 실험) / **보류** (저Impact·고Effort)
5. **TOP 3 판정** — ★ 사분면 우선, 동률이면 적합도 우위 → 자원 제약(예산·인력으로 실행 가능한가) 최종 필터. 각 순위에 "왜 이 순서인가" 1줄 + 참조 점수.
6. **1순위 90일 액션** — 30일(검증: 최소 테스트·기준 지표) / 60일(확장 판단 게이트) / 90일(본 진입 여부 판정 기준)의 3단 액션.
7. **산출 + 체인 제안** — 018 출시 체크리스트 & GTM 플랜 (1순위 시장의 런칭 타임라인 전개).

## Anti-Patterns
- **채점 후 기준 소급 수정** — 원하는 후보가 1등이 되도록 가중치를 뒤에서 바꾸면 매트릭스는 정당화 도구로 전락한다.
- **후보별 다른 잣대** — A는 매출 규모, B는 검색량으로 재면 비교가 아니다. 같은 항목·같은 척도.
- **근거 없는 점수** — 수치·가정 병기 없는 "적합도 4점"은 감이다. 091과 같은 원칙: 점수 근거 숨김 금지.
- **점수순 기계 나열** — Impact 1등이라도 Effort가 자원 제약을 초과하면 1순위가 아니다. 자원 필터까지가 판정이다.
- **90일 액션 없는 결론** — "일본부터"로 끝나면 실행이 없다. 30일 검증 테스트와 판정 지표까지 붙인다.

## 활용
- **연간 확장 계획**: 후보 채널·카테고리를 연초에 전부 올려 매트릭스로 확정 — "다음 라운드" 후보는 분기 재평가 대기열로 관리한다.
- **004와 세트 실행**: 후보마다 004로 SOM을 먼저 산출한 뒤 이 스킬로 우선순위를 매기면 Impact 축 전체가 [추정] 없이 완성된다.
- **경영진 의사결정 자료**: 채점표+매트릭스를 그대로 보고 — "왜 이 시장부터인가"에 점수 근거로 답한다.
