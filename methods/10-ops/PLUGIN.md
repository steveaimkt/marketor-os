# 10-ops — 마케팅 운영 & 생산성

## 이 카테고리가 해결하는 문제
마케터 자신의 업무를 운영체제 위에 올린다. 무엇부터 자동화할지 진단하고, 나만의 스킬을 만들고, 회의록·외주 브리프·예산·회고 같은 반복 운영 업무를 표준화한다. 종착점은 100(멀티 에이전트 오케스트레이션) — 앞의 99개 스킬을 하나의 마케팅 OS로 묶는 책의 마지막 장이다.

## 추천 진입 경로
- **대표 커맨드: /마케터OS셋업** — 체인 `091 → 092 → 100` (업무 자동화 진단 → 나만의 스킬 만들기 → 오케스트레이션 가동)
- **처음이면 091(마케팅 업무 자동화 진단)부터** — OS 온보딩의 진입점. 내 업무 리스트에서 ROI 맵이 나와야 092에서 무엇을 스킬로 만들지 정해진다.

## 다른 카테고리와의 연결
- **→ 05-ads**: 095(마케팅 예산 플래너) → 046(ROAS 진단 & 예산 재배분), 099(법무 기초 체커) → 049(광고 정책·심의 체커) — 예산과 규제 점검이 광고 실행 레이어와 왕복한다
- **→ 07-analytics**: 095(마케팅 예산 플래너) → 069(예측 시뮬레이션) — 예산 시나리오를 매출 시뮬레이션으로 검증한다
