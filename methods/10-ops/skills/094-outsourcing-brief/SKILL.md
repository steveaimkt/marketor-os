# 094 외주 브리프 빌더

"알아서 잘 해주세요"가 만드는 재작업 지옥을 끊는다. 견적서 생성 에이전트의 **요구사항 구조화 문답** 계승: 유형 선택 → 유형별 정보 수집 → 표준 문서. 범위·산출물·수정 횟수·일정·검수 기준 5대 축이 빠짐없이 문서화된 브리프만 발주된다.

## Contract
1. 브리프에는 5대 축이 반드시 명시된다: **범위**(포함/불포함) · **산출물**(파일 형식·규격·수량) · **수정 횟수**(무상 범위와 초과 단가) · **일정**(중간 시안·최종 납기) · **검수 기준**(합격 조건).
2. 레퍼런스는 링크 나열이 아니라 "이 레퍼런스의 어느 점"인지 특정해 첨부된다 (레퍼런스당 좋은 점 1줄 + 피할 점 1줄).
3. 저작권·원본 파일(작업 파일 포함) 귀속과 2차 활용 범위가 필수 항목으로 포함된다.
4. 불포함 범위(Out of Scope)가 포함 범위와 같은 비중으로 명시된다 — 분쟁의 80%는 여기서 난다.
5. 브랜드 관련 정보(톤·컬러·로고 규정)는 brand/profile.md 에서 자동 주입한다 — 사용자에게 재질문하지 않는다.

## Phases
1. **유형·범위 확정** — 외주 유형을 선택받는다: ⓐ 디자인(로고·상세페이지·배너) ⓑ 영상(촬영·편집·모션) ⓒ 개발(웹·앱·자동화) ⓓ 기타(직접 설명). 프로젝트 목적 1문장과 예산 범위·희망 납기를 함께 확정.
2. **유형별 상세 문답** — 유형에 맞는 질문 세트로 요구사항을 구조화한다 (한 번에 3문항 이내):
   - ⓐ 디자인: 산출물 규격(사이즈·해상도·플랫폼) / 사용 채널 / 원본(AI·PSD) 필요 여부 / 텍스트·카피는 누가 주나
   - ⓑ 영상: 길이·비율·용도 / 촬영 포함 여부 / 자막·BGM·내레이션 범위 / 소스 제공 목록
   - ⓒ 개발: 기능 목록(필수/선택 구분) / 참고 서비스 / 운영 환경·호스팅 / 유지보수 포함 여부
3. **표준 브리프 작성** — 10항목 골격에 배치: ①프로젝트 개요 ②배경·목적 ③범위(포함/불포함) ④산출물 명세 ⑤레퍼런스(좋은 점 특정) ⑥브랜드 가이드(profile.md 주입) ⑦일정(시안·피드백·납기) ⑧수정 정책(무상 n회 + 초과 단가 협의) ⑨검수 기준 ⑩저작권·원본 귀속.
4. **검수 기준표 작성** — 산출물별 합격 조건을 검증 가능한 문장으로: "고급스럽게" (✗) → "인쇄 시 300dpi, 로고 최소 여백 규정 준수, 오탈자 0건" (✓). 검수 회차(1차 시안/최종)별 확인 항목 분리.
5. ⏸ **발주 전 체크** — 브리프 완성본과 체크리스트(5대 축 ✓ / 불포함 명시 ✓ / 귀속 조항 ✓ / 예산-범위 정합 ✓)를 제시하고 사용자가 확정한다. 확정 후 096(견적 비교) 체인을 제안 — 같은 브리프를 복수 업체에 보내야 견적이 비교 가능해진다.

```

판정 요약: 5대 축 완비 · 산출물 {n}종 · 무상 수정 {n}회 · 납기 {날짜}

### ① 표준 브리프 (발송용)
```

## Anti-Patterns
- **"알아서 잘" 브리프** — 목적·범위 없이 레퍼런스 링크만 던지는 것. 결과물 복불복은 업체 탓이 아니라 브리프 탓.
- **수정 무제한 방치** — 수정 횟수·초과 단가를 안 정하면 무상 수정 요구와 업체 잠수 사이에서 프로젝트가 죽는다.
- **불포함 생략** — "상세페이지 디자인"에 촬영·카피가 포함인지 아닌지 안 적으면 견적도 분쟁도 커진다.
- **검수 기준 형용사** — "세련되게, 고급스럽게"는 검수가 불가능하다. 측정·확인 가능한 조건으로.
- **원본 귀속 뒷전** — 납품 후 작업 파일(AI·프로젝트 파일)을 못 받으면 다음 수정 때 같은 업체에 종속된다. 계약 전에 명시.

## 활용
- **상세페이지 외주**: 051 산출물(24섹션 기획서)을 ④산출물 명세의 입력으로 붙이면 브리프가 반나절이 아니라 10분 — 기획은 내부, 디자인만 외주.
- **협찬 영상 편집 외주**: 088 기획안의 8단 구조를 ⑤레퍼런스+구성안으로 첨부해 편집자에게 발주 — 컷 구성 재설명 불필요.
- **복수 업체 비교 발주**: 같은 브리프로 3사 견적 수취 → 096 비교 분석 → 협상 포인트 도출까지 한 체인.
