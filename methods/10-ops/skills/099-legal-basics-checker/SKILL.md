# 099 법무 기초 체커

법무팀 없는 팀의 사전 점검 1차 필터. [gates/compliance-gate.md](../../../../gates/compliance-gate.md)의 3 STEP 방법론(룰 로드 → 전수 스캔 → 판정 리포트)과 [brand/compliance.md](../../../../brand/compliance.md)의 표시광고법 룰을 광고 소재 밖 **사업 활동 전반**으로 일반화 — 표시광고법 + 개인정보보호(수집 고지·파기) + 전자상거래(청약철회)를 한 번에 점검한다.
**경계**: 099는 **사업 전반**(이벤트·수집 양식·판매 정책·환불 규정) — **광고 소재**의 매체별 심의·금지 표현 검사는 049 광고 정책·심의 체커.

## Contract
1. 점검 범위는 3법 기초로 한정한다: **표시광고법 · 개인정보보호법 · 전자상거래법**. 계약·노무·지재권 등은 범위 밖임을 산출물에 명시한다 (아는 척 금지).
2. 판정은 compliance-gate 체계를 계승한 3단 — **⛔ 고위험(즉시 수정) / ⚠️ 주의(수정 권고·근거 확보) / ✅ 이상 없음** — 항목 단위로 낸다.
3. 모든 ⛔·⚠️ 항목에 **근거(관련 법·조항 수준) + 수정안(적용 가능한 문구·절차)**을 병기한다 — 겁만 주고 대안 없는 지적 금지.
4. 산출물 첫머리와 말미에 **"법률 자문이 아니며 최종 판단은 전문가(변호사·개인정보보호 전문가) 확인 필요"** 고지를 필수 부착한다 — "적법합니다" 단정 표현 금지.
5. 업종·커스텀 룰은 profile.md §1(규제 카테고리)·§6(금기·필수 고지)과 compliance.md에서 자동 로드한다 (재질문 금지).

## Phases
1. **대상 분류 (룰 로드)** — 점검물을 영역에 매핑: ⓐ 표시·광고성 요소(가격 표시·후기·최상급) ⓑ 개인정보 요소(수집 양식·DB·이벤트 응모) ⓒ 거래 요소(판매 페이지·환불·구독). profile.md 업종 + compliance.md 해당 섹션 로드. **광고 소재 단독**이면 049로 핸드오프.
2. **표시광고법 점검** — compliance.md 공통 섹션 기준 전수 스캔: 최상급 표현(최고·1위·유일)의 근거 자료 유무 / 후기·체험단 대가성 표기 / 할인 전 가격의 실판매 이력 / 근거 없는 비교·비방. + profile §6 브랜드 금기(수익 보장 표현 등) 대조.
3. **개인정보 점검** — 수집 양식·이벤트 기준 5항: ①수집 항목 최소화(목적 대비 과다 항목 검출) ②수집·이용 목적 및 보유 기간 고지 ③동의 절차(필수/선택 분리, 마케팅 수신 동의 별도) ④파기 규정(기간 경과·목적 달성 시 파기 명시) ⑤제3자 제공·처리 위탁 고지(경품 배송 대행 등).
4. **전자상거래 점검** — 판매·거래 요소 4항: ①청약철회(7일) 안내 및 제한 사유의 적법성 ②환불 규정 표시 위치·내용(교육 서비스는 환불 기준표) ③통신판매업 신고번호·사업자 정보 표시 ④재화·서비스 정보 표시 의무 사항.
5. **판정 리포트** — 항목별 3단 판정표 + ⛔·⚠️ 건마다 {현재 상태 → 근거 → 수정안}. ⛔ 존재 시 수정안 반영본 기준 재점검 1회 (게이트의 재시도 루프 계승).
6. ⏸ **확정·체인** — 수정안 채택 여부를 사용자가 결정. ⛔ 잔존 상태로 집행 강행은 전문가 확인을 조건으로 권고. 이후 광고 소재 제작·집행 단계면 049 체인.

## Output Format
```
## 법무 기초 점검 — {대상물} · {날짜}
> ⚖️ 본 점검은 법률 자문이 아닌 사전 참고용입니다. 최종 판단은 전문가 확인이 필요합니다.
판정 요약: ⛔ {n}건 · ⚠️ {n}건 · ✅ {n}건 — 범위: 표시광고·개인정보·전자상거래 (그 외 법 영역 미점검)

### ① 3법 점검표
| 영역 | 점검 항목 | 판정 | 현재 상태 |
|---|---|---|---|
| 개인정보 | 보유 기간·파기 고지 | ⛔ | 수집 고지에 파기 시점 없음 |
| 전자상거래 | 청약철회 안내 | ⚠️ | 환불 규정은 있으나 7일 철회권 미표기 |

### ② 위험 항목 상세 (⛔·⚠️ 전건)
1. ⛔ {항목} — 근거: {관련 법·조항 수준} / 수정안: "{적용 문구}"

### ③ 전문가 확인 권고
{⛔ 잔존 건·해석 여지 건 — 확인받을 질문 형태로 정리}

다음 액션: → 049 광고 정책·심의 체커 (광고 소재 집행 전 매체별 심의 점검)
```

## Anti-Patterns
- **자문 단정** — "적법합니다 / 위법입니다"로 끝내기. 이 스킬의 언어는 "위험 신호 + 수정안 + 전문가 확인"까지다.
- **근거 없는 겁주기** — 조항 근거 없이 "큰일 난다"만 반복. 근거를 못 대는 항목은 ⚠️가 아니라 "확인 필요"로 분리.
- **049 월권** — 광고 카피의 매체 심의 통과 여부를 여기서 판정. 소재 문구는 049, 여기는 사업 구조·정책·양식.
- **동의만 받으면 끝 판정** — 동의 체크박스 유무만 보고 통과. 항목 최소화·보유 기간·파기·제3자 제공이 다 있어야 한 세트다.
- **범위 밖 즉흥 판정** — 근로계약·상표권 질문에 그 자리에서 답하기. 3법 밖은 "범위 밖 + 전문가 영역" 명시가 정답이다.

## 활용
- **이벤트 오픈 전 점검**: 응모 양식(개인정보 5항) + 경품 고지(표시광고) + 당첨자 배송 위탁(제3자 제공)을 오픈 전날 한 번에 — 사후 삭제 공지보다 백배 싸다.
- **모집 페이지 정기 점검**: 교육 모집 페이지의 환불 기준표·수익 보장성 표현·후기 표기를 기수마다 점검 — profile §6 금기("수강하면 매출 오릅니다" 금지)가 자동 대조된다.
- **049와 이중 게이트**: 신규 캠페인 = 099(사업 구조·양식) 선점검 → 소재 제작 → 049(매체 심의) 후점검 — 기획 단계와 집행 단계의 리스크를 각각 끊는다.

> 참고 — 3법 1차 필터 암기용 요약 (상세 점검 항목은 Phases 2~4):
> - 표시광고법: 최상급엔 근거 · 후기엔 대가성 표기 · 할인가엔 실판매 이력 · 비교엔 자료
> - 개인정보: 최소 수집 · 목적/보유 기간 고지 · 필수/선택 동의 분리 · 파기 규정 · 제3자 제공/위탁 고지
> - 전자상거래: 청약철회 7일 · 환불 규정 표시 · 통신판매업 신고·사업자 정보 표시
