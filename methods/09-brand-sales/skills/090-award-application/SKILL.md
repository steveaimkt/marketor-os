# 090 수상·인증 지원서 빌더

> 정부지원사업·어워드·인증 지원서를 **평가 기준 역산**으로 쓴다. 우리가 하고 싶은 말 순서가 아니라, 심사위원이 점수를 매기는 순서로.
> 계승: 084 제안서 자동 생성의 검증 파이프라인(공고 분석 → Win Theme → 역산 목차 → Action Title) 응용 — RFP가 공고문으로, 고객이 심사위원으로 바뀔 뿐 뼈대는 같다. 지원서 특유의 추가 계약: 증빙 목록과 분량 제약 대응.

## Contract
- 초안 작성 전에 공고문에서 **평가 기준·배점·자격 요건·분량 제약·제출 서류 목록**을 추출한다 — 배점표 없이 본문을 쓰지 않는다 (공고에 배점이 없으면 유사 사업 통례 배점을 [추정]으로 세우고 명시).
- 자격 요건 미충족이 발견되면 그 시점에 멈추고 보고한다 — 못 낼 서류를 정성껏 쓰는 낭비를 차단.
- 모든 성과 주장에는 증빙이 매핑된다 — 증빙 없는 주장은 본문에서 빼거나 `[증빙 필요]` 태그로 확보 방법과 함께 남긴다. 허위·과장 기재는 게이트에서 차단 (선정 취소·환수 사유다).
- 항목별 분량 제약(글자수·페이지)을 준수한 초안을 낸다 — 제한 초과분은 "삭제 우선순위"를 붙여 잘라낸다.
- 선정 논리(Win Theme) 3개가 먼저 확정되고, 모든 항목 초안에 어느 논리를 뒷받침하는지 태깅된다 (084 방식).

## Phases
1. **공고 분석** — 공고문에서 추출: 사업 목적(돈을 주는 이유) · 평가 기준·배점 · 자격 요건 · 항목별 분량 제약 · 제출 서류 목록 · 마감·일정. [brand/profile.md](../../../../brand/profile.md) §1·§3 자동 참조.
2. **적합성 판정** — 자격 요건 체크리스트 (업력·매출·업종·중복 수혜 제한). 미충족 → 여기서 멈추고 대안 공고 방향 제안. 충족 → 진행.
3. **선정 논리 도출** ⏸ — 최고 배점 기준 ↔ 우리 강점을 교차해 선정 논리 3개 작성: "평가 기준 {X}에 우리는 {근거·수치}로 답한다" + 지지 증빙 1개씩. 사업 목적과의 정렬(이 돈이 왜 우리에게 오면 성과가 나는가)을 1줄로 요약. 사용자 승인 후 본문 진행.
4. **역산 구조 설계** — 지원서 항목(현황·계획·성과·사업화 등)마다 ① 대응 평가 기준·배점 ② 배점 비례 분량 배분 ③ Action Title(문장형 소제목 — "기업 현황"이 아니라 "3년간 재구매율 41%를 만든 운영 구조") ④ 대응 선정 논리 번호를 배정한다.
5. **초안 작성 + 증빙 목록** — 항목별로 주장→수치→증빙 참조 번호(예: [증빙3]) 구조로 집필. 동시에 증빙 목록 표 작성: 문서명 · 무엇을 입증 · 확보 상태(✅ 보유 / 🔄 발급 필요+소요일 / ❌ 미확보). 수치는 출처·기준 시점 병기, 추정치는 [추정].
6. **분량 검사** — 항목별 글자수 카운트 → 제한 초과 시 삭제 우선순위(선정 논리 태깅 없는 문장부터)로 압축, 미달 시 증빙 기반 보강 후보 제시.
7. ⏸ **compliance-gate 통과** — 과장·허위 소지 표현("국내 유일", "보장") 전수 검사 + 증빙 없는 주장 잔존 여부 확인. 판정 블록 부착본만 전달 → 체인: 084 제안서 자동 생성 (같은 선정 논리·증빙으로 B2B 제안서 파생).

## Anti-Patterns
- **회사 자랑 순서로 쓰기** — 연혁부터 시작하는 지원서는 배점표를 안 읽었다는 자백이다. 최고 배점 항목이 가장 두꺼워야 한다.
- **증빙 없는 성과 수치** — "매출 300% 성장"을 재무제표 없이 쓰면 서면 심사는 통과해도 현장 실사에서 무너진다. 주장 수 = 증빙 매핑 수.
- **분량 제한 무시 초안** — 2,000자 제한에 4,000자를 주고 "알아서 줄이세요"는 미완성 납품이다. 제한 준수본이 계약.
- **사업 목적 미정렬** — 수출 지원 사업에 내수 성장 계획을 쓰는 것. 돈을 주는 이유에 우리 계획을 정렬하지 못하면 논리가 아무리 좋아도 탈락한다.
- **마감 역산 없는 증빙 계획** — 발급에 2주 걸리는 인증서를 마감 3일 전에 발견하는 것. 증빙 목록에 소요일을 붙이는 이유다.

## 활용
- **정부지원사업 시즌 대응**: 1~3월 공고 러시에 공고문만 바꿔 재실행 — 선정 논리·증빙 목록이 자산으로 쌓여 회차마다 빨라진다.
- **어워드·인증 브랜딩**: 수상 실적을 083 보도자료·085 세일즈 덱의 증거 슬라이드로 연결 — 지원서 1회가 마케팅 자산 3개가 된다.
- **입찰 파생**: 확정된 선정 논리·증빙 패키지를 084에 넘겨 공공 입찰 제안서로 전환 — /세일즈파이프라인 체인과 합류한다.
