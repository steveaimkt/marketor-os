# 089 위기 커뮤니케이션 플랜

이슈가 터진 순간 가장 위험한 건 침묵이 아니라 **성급한 확정 발언**이다. 이 스킬은 이슈를 유형·심각도로 분류하고, 대응/무대응/에스컬레이션 의사결정 트리를 태운 뒤, 채널별 대응 문구를 게이트 통과본으로 낸다. 모든 문구는 [brand/profile.md](../../../../brand/profile.md) §6 금기(보장·과장·공포 조장 금지)를 준수한다.

## Contract
- **사실 확인 전에는 사과도 부인도 확정하지 않는다** — 초기 대응은 홀딩 스테이트먼트(인지+확인 중+시한 약속)만 낸다.
- 문구 작성 전에 유형(품질/CS/법적/여론) × 심각도(1~3) 분류가 선행된다 — 분류 없는 문구 요청은 분류부터 되돌린다.
- **법적 유형은 이 스킬이 종결하지 않는다** — 모든 산출물에 `[변호사 검토 필수]` 플래그를 붙이고, 책임 인정 표현을 초안에서 배제한다.
- 무대응도 정식 선택지다 — 대응/무대응 판단 기준(확산 속도·매체 급·사실 여부)을 수치·조건으로 명시하고, 무대응 선택 시 모니터링 조건을 함께 정의한다.
- 모든 대외 문구는 compliance-gate 3 STEP + profile.md §6 검사를 통과한 판정 블록과 함께만 전달된다.

## Phases
1. **사실관계 정리** — 타임라인 작성: ✅ 확인된 사실 / ❓ 미확인 주장 / 🗣 여론 반응을 분리. 이 구분이 모든 문구의 수위를 결정한다. profile.md §6 금기·§3 포지셔닝 자동 로드.
2. **유형·심각도 분류** — 유형: 품질(제품 하자·안전) / CS(응대 실수·개인 건 확산) / 법적(표시광고법·계약·저작권) / 여론(가치관·설화·오해). 심각도: L1 단일 채널·소규모 / L2 커뮤니티 확산·언론 문의 전 / L3 언론 보도·법적 통보·매출 영향. 복합 유형이면 가장 무거운 쪽 기준.
3. **의사결정 트리** — 분기 기준 명시: ① 사실인가? (사실 → 인정+조치 트랙 / 허위 → 정정 트랙 / 미확인 → 홀딩 트랙) ② 확산 중인가? (24시간 내 언급 증가율·매체 급으로 판정 — 자연 소멸 추세면 무대응+모니터링) ③ L3 또는 법적 유형인가? (→ 에스컬레이션: 대표 승인·변호사 검토 라인 가동) ⏸ 사용자가 트리에서 대응 수위를 확정한다.
4. **채널별 문구 작성** — 확정 트랙에 따라: 홀딩 스테이트먼트(전 채널 공통 1종) → 공지문(자사 채널) → 댓글·DM 응대 스크립트(032 연동 형식) → 언론 답변서(L3). 사과문은 4단 구조 고정: 인정(구체적으로 무엇을) → 사과(조건 없이 — "불편하셨다면" 금지) → 조치(이미 한 것+할 것, 날짜) → 재발 방지(구조 변경). 공포 조장·책임 전가·과장 수습 표현 금지.
5. ⏸ **compliance-gate 통과** — [gates/compliance-gate.md](../../../../gates/compliance-gate.md) 3 STEP + §6 금기 검사 + (법적 유형) `[변호사 검토 필수]` 플래그 확인. 판정 블록 부착본만 전달.
6. **사후 체계·체인** — 위기 키워드 목록(이번 이슈에서 도출)을 032 댓글·DM 응대 어시스턴트의 위기 키워드 감지 사전에 등록 제안 → 주 체인: 032 (일상 응대 감시 체계로 전환) · 083 보도자료 (L3 진화 후 공식 입장·개선 발표).

## Anti-Patterns
- **선사과 후확인** — 사실 확인 전에 전면 사과부터 내면, 허위 제보로 판명돼도 인정 기록이 남는다. 홀딩이 먼저다.
- **조건부 사과** — "불편을 느끼셨다면 사과드립니다"는 사과가 아니라 도발이다. 사과 트랙이면 조건 없이, 아니면 사과 단어를 쓰지 않는다.
- **전 채널 동일 문구 살포** — 공지문을 댓글에 복붙하면 기계 대응으로 읽혀 2차 화상을 만든다. 채널별 수위·길이 분리.
- **법적 이슈 셀프 종결** — 표시광고법·계약 분쟁 문구를 게이트 통과만으로 내보내는 것. [변호사 검토 필수]는 장식이 아니라 차단선이다.
- **과잉 대응** — L1 단일 불만에 대표 명의 사과문을 내면 이슈를 스스로 승격시킨다. 무대응+모니터링도 전략이다.

## 활용
- **평시 예방 훈련**: 이슈가 없을 때 자사 취약 시나리오 3개(품질·CS·여론)로 모의 플랜을 만들어 둔다 — 위기 시 4단계부터 시작할 수 있다.
- **032 감시 체계 연동**: 도출된 위기 키워드를 032의 감지 사전에 상시 등록 — 다음 이슈는 댓글 단계(L1)에서 잡힌다.
- **협업·인플루언서 리스크**: 034 인플루언서 계약 건에서 설화가 났을 때 여론 유형 트리로 즉시 분류 — 손절/유지 판단 기준이 감정이 아니라 트리가 된다.

> 부록 — 유형 × 심각도 초기 대응 매트릭스 (2단계 분류 직후 참조)

| 유형 \ 심각도 | L1 | L2 | L3 |
|---|---|---|---|
| 품질 | 개별 보상 + 원인 확인 | 홀딩 + 공지 준비 | 공지 + 회수/조치 발표 [대표 승인] |
| CS | 당사자 직접 사과 | 공개 댓글 + 재발 방지 | 공지문 [대표 승인] |
| 법적 | [변호사 검토 필수] — 전 레벨 공통 | 동일 | 동일 + 대응 창구 일원화 |
| 여론 | 무대응 + 모니터링 우선 | 입장 정리 1회 | 공지 + 083 보도자료 |
