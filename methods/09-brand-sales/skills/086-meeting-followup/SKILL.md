# 086 미팅 노트 → 후속 액션

**대외 미팅**(고객 상담·영업·파트너 협의) 녹취와 메모를 ① 결정사항 ② 할 일(담당·기한) ③ 팔로업 메일 초안 3종으로 분리해 낸다. 미팅이 끝난 지 24시간 안에 메일이 나가야 수주 온도가 식지 않는다 — 그 속도를 만드는 스킬.
**093과 경계**: 086 = 대외 미팅·영업 후속 (상대가 고객·파트너, 산출물이 메일로 나감) / 093 = 내부 회의 위임 (상대가 팀, 산출물이 태스크 배분). 내부 회의록이면 093으로.

## Contract
- 요약은 **결정사항 / 보류·미결 / 할 일** 3구획으로 분리한다 — "논의했다"와 "결정했다"를 섞지 않는다.
- 할 일마다 담당·기한을 특정한다 — 녹취에 없으면 지어내지 않고 `[담당 미정]`·`[기한 미정]`으로 표기해 확인을 유도한다.
- 녹취에 없는 합의·약속을 창작하지 않는다 — 불명확한 발언은 `[확인 필요: 원문 "…"]`으로 원문 발췌와 함께 남긴다.
- 팔로업 메일은 24시간 내 발송 가능한 완성도로 내고, **미팅 중 상대 발언 인용을 1회 이상** 포함한다 (복붙 티 나는 범용 메일 금지).
- 이 스킬은 초안까지다 — 메일 발송·일정 등록 등 외부 실행은 하지 않는다 (mutating: false).

## Phases
1. **입력 수신** — 녹취 전사본 또는 메모 + 참석자·미팅 목적. [brand/profile.md](../../../../brand/profile.md) §1 서비스·§3 포지셔닝 자동 참조 (메일 서명·제안 방향 근거). 내부 회의로 판별되면 093으로 안내하고 종료.
2. **3구획 추출** — 발언을 분류: ✅ 결정사항(양측이 합의한 것 — 근거 발언 위치 표기) / ⏳ 보류·미결(다음 접점으로 넘어간 것 + 막힌 이유) / ☑️ 할 일(우리 것 vs 상대 것 분리).
3. **할 일 표 작성** — 항목별 담당·기한·완료 조건. 우리 할 일과 상대 대기 항목(우리가 기다리는 것)을 분리해 팔로업 리듬을 설계한다.
4. **영업 신호 스캔** — 상대 발언에서 Pain Point·예산 언급·일정 압박·경쟁 검토 신호를 추출해 별도 메모로 (084 제안서의 분석 입력으로 그대로 넘어가는 재료). 신호 없으면 "신호 없음"으로 정직하게.
5. **팔로업 메일 초안** — 구조: 감사 1줄 → 결정사항 요약 3줄 이내 → 상대 발언 인용 1회(맞춤성 증명) → 우리가 보낼 것/할 것 → 다음 스텝 제안(일시 2안 제시). 제목 2안. 톤은 [brand/tone.md](../../../../brand/tone.md) 준수.
6. ⏸ **확인·체인** — `[담당 미정]`·`[확인 필요]` 항목을 사용자에게 확인받아 반영 → 수주 가능성이 확인된 건이면 084 제안서 자동 생성으로 핸드오프 (영업 신호 메모가 Pain Point 입력이 된다).

### ④ 팔로업 메일 초안 (제목 2안 + 본문 — 인용 1회 포함)

## Anti-Patterns
- **회의록 전문 재생산** — 발언 순서대로 다 적은 문서는 아무도 다시 안 읽는다. 결정·보류·할 일만 남기고 버린다.
- **합의 창작** — "긍정적으로 검토하겠다"를 "도입 결정"으로 승격시키는 것. 결정사항에는 근거 발언이 반드시 붙는다.
- **담당·기한 임의 배정** — 미팅에서 안 정한 기한을 그럴듯하게 채우면 상대와의 약속이 어긋난다. [미정] 표기가 정직한 산출물.
- **범용 감사 메일** — "좋은 시간이었습니다. 잘 부탁드립니다"는 안 보낸 것과 같다. 상대 발언 인용 + 구체 다음 스텝이 없으면 미완성.
- **내부 회의 침범** — 팀 주간회의 정리를 이 스킬로 하면 영업 신호 스캔이 헛돈다. 대외/내부 판별 후 093과 분업.

## 활용
- **영업 상담 직후 루틴**: 미팅 종료 → 녹취 붙여넣기 → 당일 저녁 메일 초안 확정 → 익일 오전 발송 — /세일즈파이프라인 체인(086→084→085)의 시작점.
- **파트너 협의 추적**: 보류·미결 목록을 다음 미팅 아젠다로 그대로 재사용 — "지난번에 어디까지 얘기했죠"가 사라진다.
- **상대 대기 항목 관리**: "상대 할 일" 표의 기한 경과 항목만 모아 리마인드 메일 초안을 재생성 — 회신 없는 딜의 온도를 유지한다.
- **미팅 자산 축적**: 영업 신호 메모를 고객별로 누적하면 재접촉 시점에 085 덱의 고객 유형 판정(신규/전환/갱신)이 데이터로 정해진다.

> 부록 — 086 vs 093 판별 기준 (라우팅 경합 방지)

| 신호 | 판정 |
|---|---|
| 참석자에 고객·파트너·외부인 포함, 산출물이 상대에게 발송됨 | 086 (이 스킬) |
| 참석자 전원 내부, 산출물이 팀 태스크 배분 | 093 회의록·업무 위임 |
| 내부 회의지만 특정 고객 딜 논의 → 후속이 대외 메일 | 086 |
