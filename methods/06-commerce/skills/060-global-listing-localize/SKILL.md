# 060 글로벌 리스팅 로컬라이즈

> 051 산출물(24섹션 기획서)을 입력으로 영/일 마켓 리스팅으로 변환 — **직역 금지·소구점 재구성** + 규격·단위·인증 변환 + 아마존/쇼피 필드 제약 준수.
> 계승: 051 상세페이지 24섹션 산출물 기반 + 멀티 콘텐츠 생성기의 채널 변형 로직(원본 1개→포맷별 재구성)을 마켓 변형으로 전용.

## Contract
- **직역 금지** — 소구점을 마켓 관용으로 재구성하고, 섹션마다 [재구성/직역 허용/삭제] 결정 로그를 남긴다 (재구성 사유 병기).
- 단위·규격 변환표를 필수 동봉한다 (ml↔fl oz, cm↔inch, g↔oz, 전압·플러그, 인증) — 변환 수치는 산식과 함께 표기하고 임의 반올림으로 규격을 다르게 적지 않는다.
- 플랫폼 필드 제약을 준수한다 — 제한 초과분은 **자동 절삭이 아니라 재작성**한다.
- 마켓별 금지 표현·필수 고지를 게이트에서 검사한다 (미국: FDA 의약적 효능 표현 / 일본: 약기법(薬機法) 효능 범위) — 통과 후에만 전달 + 치환 로그.
- 원본이 없으면 변환하지 않는다 — 051 상세페이지 24섹션 빌더 선행을 권고한다.

## Phases
1. **원본·마켓 확인** — 051 산출물(24섹션 기획서) 우선 로드, 없으면 기존 페이지 텍스트 접수 후 섹션 대조. 대상 마켓 확정: US 아마존 / JP 아마존 / 쇼피(동남아) 등. profile.md에서 제품 규격·인증 보유 현황 참조.
2. **소구점 재구성** — 한국형 소구를 마켓 소구 맵으로 변환:

| 원본 소구 (KR) | US 재구성 | JP 재구성 |
|---|---|---|
| 감성 공감형 ("이런 고민 있으시죠") | 기능·수치 선행 (문제 해결 결과 명시) | 불안 해소·안심 (저자극·성분 안전) |
| 권위 어필 ("판매 1위") | 리뷰·평점 근거 (증빙 가능 시만) | 수상·인증 이력 (문서 병기) |
| 트렌드 소구 ("요즘 대세") | 구체 사용 시나리오 | 정중체·과장 억제 톤 |

3. **규격·단위 변환표** — 용량(ml→fl oz ÷29.5735)·치수(cm→inch ÷2.54)·중량(g→oz ÷28.3495)·전압(220V→110V 호환 여부·플러그 타입)·인증(KC는 해외 무효 → US: FDA 등록/FCC, JP: PSE/약기법 구분 표기). 미보유 인증은 "취득 필요"로 플래그.
4. **플랫폼 필드 제약 적용** —

| 필드 | 아마존(US/JP) | 쇼피 |
|---|---|---|
| 제목 | 200자 권장 80~120 (모바일 노출 기준) | 120자 |
| 핵심 소구 | 불릿 포인트 5개 (각 200자 내) | 상세 설명 상단 5줄 |
| 검색 키워드 | 백엔드 키워드 250바이트 (경쟁 브랜드명 금지) | 해시태그 |
| 이미지 | 메인 흰 배경·텍스트 금지 | 메인 1:1 |

5. **카피 현지화** — 섹션별 재구성 작성 + 결정 로그(원문→변환문→[재구성/직역/삭제]+사유). 날짜·주소·고객센터 표기도 마켓 관행으로 (MM/DD/YYYY, 현지 반품 정책 문구).
6. ⏸ **규제 게이트** — 마켓 규제 사전으로 gates/compliance-gate.md 검사 (US: 의약적 효능·whitening류 표현 / JP: 약기법 효능 56개 범위 외 표현) → 치환 로그 부착 후 전달. 리워드: 마켓 등록은 사용자가 셀러센트럴에서 수동 실행.

## Output Format
```
## 글로벌 리스팅 — {제품명} · {US 아마존|JP 아마존|쇼피} · [원본: 051 기획서|기존 페이지]
판정 요약: 재구성 {n}섹션 · 직역 {n} · 삭제 {n} · 인증 플래그 {n}건 · 게이트 {✅|⚠️|⛔}

### ① 현지화 리스팅 (제목 → 불릿 5개 → 상세 설명 → 백엔드 키워드 — 플랫폼 필드 순서)
### ② 규격·단위 변환표 (항목·원본값·변환값·산식·인증 상태)

### 규제 치환 로그 (검출어·위치·대체어·근거: FDA/약기법/플랫폼 정책)

─── 🛡 컴플라이언스 게이트 ─── 판정: ✅ / ⚠️ ({n}건 치환) / ⛔ ───

다음 액션: 원본 리뉴얼이 먼저 필요하면 → 051 상세페이지 24섹션 빌더 / 등록 후 상품명 튜닝은 053 상품명·검색 최적화
```

## Anti-Patterns
- **24섹션 통짜 직역** — 아마존 리스팅은 불릿 5개+A+ 콘텐츠 문법으로 페이지 구조 자체가 다르다. 구조를 옮기지 말고 소재를 재배치.
- **한국 맥락 소구 직역** — "국민 토너"·"올리브영 1위"는 해외 고객에게 무의미하거나 증빙 불가 클레임이 된다.
- **KC 인증 그대로 표기** — KC는 해외 마켓에서 무효. 마켓별 유효 인증으로 교체하거나 취득 필요 플래그.
- **효능 표현 직역** — "미백"→"whitening"은 US OTC 규제, "주름 개선" 직역은 일본 약기법 저촉 후보. 게이트 없는 효능 번역 금지.
- **백엔드 키워드에 경쟁 브랜드명** — 아마존 정책 위반으로 리스팅 정지 사유. 검색 키워드는 일반명사·속성어로만.

## 활용
- **US 아마존 첫 진출**: 051 기획서 → 본 스킬로 리스팅 변환 → 인증 플래그 목록이 곧 진출 준비 체크리스트가 된다.
- **일본 쇼핑몰 동시 전개**: 같은 원본으로 JP 재구성 열만 갈아끼워 아마존 재팬·큐텐 병행 등록 — 결정 로그가 마켓별 차이의 관리 대장.
- **현지화 품질 검수**: 대행사 번역본을 ③ 결정 로그 형식으로 역검수 — "직역인데 재구성으로 표기된" 구간을 걸러내는 기준표.
