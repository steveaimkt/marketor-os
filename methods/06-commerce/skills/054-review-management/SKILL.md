# 054 리뷰 관리 시스템

리뷰 수십 건을 넣으면 감정 분류 → 브랜드 톤 일관 답변 초안 → 부정·위험 리뷰 에스컬레이션까지 한 번에. 답변은 초안까지만 — 게시(등록)는 항상 사람이 한다.

## Contract
1. 입력된 모든 리뷰는 **긍정 / 중립 / 부정 / 위험** 4단계 + 토픽 태그로 빠짐없이 분류된다.
2. 모든 답변 초안은 brand/tone.md 의 톤·어휘·금기어를 따른다 — 리뷰 100건이어도 톤은 1개.
3. 부정·위험 리뷰는 자동 답변 대상에서 제외되고 반드시 에스컬레이션 리스트로 분리된다.
4. 모든 답변은 compliance-gate 전수 스캔을 통과한 뒤에만 전달된다 (리뷰 답변도 대외 발행물).
5. 동일 문장 복붙 답변은 만들지 않는다 — 리뷰의 구체 표현을 1개 이상 인용해 개별화한다.

## Phases
1. **브랜드 컨텍스트 로드** — brand/profile.md(업종·제품·금기)와 brand/tone.md 를 자동 참조한다. 사용자에게 브랜드·톤을 재질문하지 않는다.
2. **리뷰 수집·정규화** — 텍스트/CSV/캡처를 `{번호, 별점, 리뷰 원문, 채널}` 표로 정규화. 개인정보(주문자명·전화)는 마스킹.
3. **감정 분류** — 각 리뷰를 4단계로 판정:
   - 긍정(칭찬·만족) / 중립(단순 수령·질문) / 부정(불만·환불 요구) / 위험(법적 언급, 안전·부작용 주장, 언론·신고 예고)
   - 토픽 태그 병기: 품질 · 배송 · 가격 · CS · 사용법 (VoC 축적용 → 006 연계)
4. **답변 초안 생성** — 긍정·중립만 자동 생성. 공통 골격: ① 구체 인용 감사 ② 리뷰 포인트 응답 ③ 브랜드 한 줄 ④ 재방문 인사. 금지: 효능 재주장, 보상 약속, 경쟁사 언급.
5. ⏸ **부정·위험 에스컬레이션** — 부정 리뷰는 "사실관계 확인 필요 항목 + 대응 방향 2안(사과·보상형 / 안내·해명형)"으로 정리해 사용자 판단을 받는다. 위험 리뷰는 답변 초안을 만들지 않고 대응 가이드만 제시한다.
6. ⏸ **compliance-gate 통과** — gates/compliance-gate.md 3 STEP(룰 로드 → 전수 스캔 → 판정 리포트)을 실행하고, 게이트 결과 블록을 부착한 최종본만 전달한다. ⛔ 판정 답변은 전달하지 않는다.

## Output Format
```
## 리뷰 답변 리포트 — {날짜} · 총 {N}건
판정 요약: 긍정 {n} · 중립 {n} · 부정 {n} · 위험 {n} → 자동 초안 {n}건 / 에스컬레이션 {n}건

### ① 감정 분류표
| # | 별점 | 리뷰 요약 | 감정 | 토픽 | 처리 |
|---|---|---|---|---|---|

### ② 답변 초안 (복붙 가능)
**#3 · ⭐5 · 긍정(품질)**
> {답변 전문 — 리뷰 구체 표현 인용 포함}

### ③ 에스컬레이션 리스트 (사람 판단 필요)
| # | 감정 | 쟁점 | 확인할 사실 | 대응 방향 A / B |
|---|---|---|---|---|

─── 🛡 컴플라이언스 게이트 ───
판정: ✅ / ⚠️ ({n}건 치환) / ⛔
검출·치환 내역: ...
──────────────────────────

다음 액션: 이번 리뷰 태그를 006 소비자 리뷰 마이닝에 넘기면 페인포인트 TOP 5가 나옵니다.
```

## Anti-Patterns
- **전 리뷰 동일 답변 복붙** — "소중한 리뷰 감사합니다"만 반복하면 안 하느니만 못하다. 반드시 리뷰 문장을 인용해 개별화.
- **부정 리뷰에 즉답** — 사실관계 확인 없이 사과·해명 초안을 바로 게시용으로 내보내지 않는다 (⏸ 필수).
- **보상 임의 약속** — "재구매 시 사은품 드릴게요" 류의 보상·환불 조건 제시는 사용자 승인 없이 금지.
- **효능 재주장** — 답변에서 "피부가 재생되죠" 처럼 리뷰의 효능 표현을 받아 확언하면 광고 규제 위반 (게이트 차단 사유).
- **위험 리뷰 직접 응대** — 법적·안전 이슈 리뷰에 스킬이 답변을 생성하는 것 자체가 리스크. 가이드만 제공.

## 활용
- **주간 리뷰 일괄 처리**: 스마트스토어 주간 신규 리뷰 30건을 매주 금요일 한 번에 → 초안 검토 10분으로 단축.
- **부정 리뷰 조기 경보**: 위험 분류가 1건이라도 나오면 그날 처리 — 에스컬레이션 리스트가 CS 우선순위 큐가 된다.
- **VoC 자산화**: 매회 분류표의 토픽 태그를 누적하면 006 소비자 리뷰 마이닝의 입력이 되어 답변 업무가 제품 개선 데이터로 전환된다.
