# 055 스마트스토어 진단

> 스토어 컨설팅(SBA 프레임워크 계열)의 진단 접근을 노출·전환·재구매 3축으로 표준화한 스킬.

"왜 안 팔리지?"를 감이 아니라 3축 프레임으로 분해한다: **노출(검색 유입) → 전환(상세페이지) → 재구매(리뷰·알림)**. 각 축을 점수화해 병목 1개를 특정하고, 2주 안에 실행 가능한 액션부터 담당 스킬과 함께 내놓는 컨설팅형 진단 스킬이다.

## Contract
1. 진단은 노출·전환·재구매 3축을 모두 다룬다 — 데이터가 없는 축은 [데이터 없음]으로 명시하고 수집 방법(스마트스토어센터 통계 메뉴 경로)을 안내한다.
2. 각 진단 항목에 현재 수치와 기준치를 병기한다 — 실측이 없으면 [추정] 태그로 구분하고 근거를 적는다.
3. 병목은 반드시 1개 축으로 특정하고 판정 근거 수치를 제시한다 ("전부 문제" 결론 금지).
4. 개선 액션은 2주 내 실행 가능한 것부터 정렬하고, 각 액션에 담당 스킬 ID를 연결한다 (노출→053, 전환→051, 광고→042, 리뷰→054).
5. 진단은 읽기 전용이다 — 스토어 설정·상품 정보를 직접 변경하지 않으며, 변경은 연결된 스킬에서 사용자가 실행한다.

## Phases
1. **지표 취합** — 스마트스토어센터 통계(유입수·유입 채널·결제 전환율·재구매율)를 입력받는다. 수치 미제공 시 스토어 URL 기반 체크리스트 정성 진단 모드로 전환 (모든 판정에 [추정] 태그).
2. **노출 진단 (검색 유입 축)** — 대표 키워드 검색 순위, 상품명·태그의 키워드 적합도, 카테고리 정확도, 유입 채널 구성(검색/광고/외부 비중). 유입 자체가 적으면 이 축이 1차 용의자.
3. **전환 진단 (상세 축)** — 결제 전환율을 카테고리 통상 범위와 대조, 상세페이지 첫 화면 훅·가격 경쟁력·리뷰 수·배송 조건(오늘출발 여부)을 점검. 유입은 있는데 결제가 없으면 이 축.
4. **재구매 진단 (리뷰·알림 축)** — 리뷰 응답률·평점 추이, 알림받기(스토어찜) 고객 수와 쿠폰 발행 여부, 재구매율. 첫 구매만 있고 반복이 없으면 이 축.
5. **병목 특정** — 3축 점수표(각 5점 만점)를 만들고 최저 축을 병목으로 판정, 근거 수치 2개 이상 인용.
6. **액션 목록** — 병목 축 개선 액션을 2주 실행 가능 순으로 3~5개, 각 액션에 스킬 연결. 다음: 노출 병목이면 053, 광고 유입 병목이면 042.

## Output Format
```

데이터 기준: {실측 지표 목록} / [추정] {항목}

### ① 3축 점수판
| 축 | 점수(/5) | 핵심 수치 | 기준치 대비 |
|---|---|---|---|
| 노출 (검색 유입) | | 월 유입 {n} · 검색 비중 {n}% | |
| 전환 (상세) | | 결제 전환율 {n}% | 카테고리 통상 {범위} |
| 재구매 (리뷰·알림) | | 재구매율 {n}% · 알림받기 {n}명 | |

### ② 병목 판정
병목: {축} — 근거: {수치 1} · {수치 2}

### ③ 2주 실행 액션 (우선순위순)
| # | 액션 | 기대 효과 | 담당 스킬 |
|---|---|---|---|
| 1 | {액션} | {축 지표 개선} | 053 |

### ④ 데이터 수집 안내 ([데이터 없음] 축이 있을 때)
스마트스토어센터 > 통계 > {메뉴 경로} — 다음 진단 전 확보할 지표 목록

다음 액션: 병목이 노출이면 053 상품명·검색 최적화, 광고 유입이면 042 네이버 쇼핑검색광고 최적화로 이어가세요.
```

## Anti-Patterns
- **"전부 개선하세요" 리포트** — 3축 모두에 액션을 뿌리면 아무것도 실행되지 않는다. 병목 1개 특정이 이 스킬의 존재 이유.
- **수치 없는 훈수** — "상세페이지를 매력적으로"류의 조언. 모든 판정에는 수치 근거가 붙어야 한다.
- **추정을 실측처럼** — URL만 보고 낸 정성 판정을 통계 수치처럼 단정하는 것. [추정] 태그 규칙 준수.
- **광고 진단 월권** — 광고 계정 딥다이브는 042의 영역. 여기서는 유입 구성 확인까지만 하고 넘긴다.
- **재구매 축 생략** — 데이터가 잘 없다는 이유로 3축 중 하나를 조용히 빼는 것. 없으면 [데이터 없음] + 수집법 안내.

## 활용
- **월간 정기 건강검진**: 매월 초 지난달 통계로 3축 점수판을 갱신하면 병목 이동(노출→전환→재구매)이 성장 단계 지표가 된다.
- **컨설팅 첫 미팅 자료**: 대행사·컨설턴트에게 의뢰하기 전 셀프 진단 리포트를 만들어 가면 논의가 액션 단위에서 시작된다.
- **신규 입점 90일 플랜**: 입점 직후엔 노출 축만 집중 → 유입 안정 후 전환 축 → 리뷰 쌓인 뒤 재구매 축 순으로 이 진단을 로드맵으로 사용.
- **다점포 비교 진단**: 스토어가 2개 이상이면 같은 3축 점수판으로 나란히 진단 — 점수 차가 큰 축이 곧 스토어 간 노하우 이전 항목이 된다.
