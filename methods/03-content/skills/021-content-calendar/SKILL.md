# 콘텐츠 캘린더 빌더

> 캠페인·트렌드·VoC·키워드 4입력 → 채널 믹스 배분 → 슬롯마다 근거가 붙은 월간 캘린더 (전 슬롯 Draft).
> 계승: 실습 5-4 content-calendar의 4소스 통합·채널 믹스 상수·Draft 이중 안전망 + marketing-os `content-planning`의 사업 국면 정렬.

## Contract
- 모든 슬롯에 날짜·채널·포맷·제목 가안·퍼널 단계·기획 근거·실행 스킬 7항목이 채워진다 — 빈칸 슬롯 금지.
- 슬롯 총수는 사용자가 확정한 월 발행 캐파와 정확히 일치하고, 채널 믹스 합계와 맞아떨어진다.
- 슬롯마다 기획 근거는 4입력(캠페인/트렌드/VoC/키워드) 중 최소 1개를 명시한다 — "주관"이 아니라 "근거"로 글감을 정한다.
- 소스 데이터가 없는 근거는 [추정] 태그를 붙인다.
- 전 슬롯 상태는 Draft — 이 스킬은 기획까지. 발행·예약·큐잉은 하지 않는다.

## Phases
1. **브랜드 컨텍스트 + 사업 국면** — [brand/profile.md](../../../../brand/profile.md) + [brand/tone.md](../../../../brand/tone.md) 자동 참조. 지금이 모집기/판매기/브랜딩기인지 확인 — 국면에 따라 채널 비중과 퍼널 배분이 달라진다.
2. **입력 4소스 수집** — ① 캠페인 일정(사용자 제공) ② 트렌드(005 산출물) ③ VoC(006·078 산출물) ④ 키워드(001 산출물). 미보유 소스는 브랜드 컨텍스트 기반 추정 모드로 채우고 해당 근거에 [추정] 태그.
3. **채널 믹스 확정** ⏸ — 운영 채널×주간 캐파를 상수로 선언 (예: IG 7 · Blog 4 · LinkedIn 4 · Threads 8 · Newsletter 2 = 25). 사용자 확인 후 고정 — 이후 임의 변경 금지.
4. **퍼널 배분** — TOFU(인지) 50% / MOFU(고려) 30% / BOFU(전환) 20% 기준으로 국면에 맞게 조정. 캠페인 D-Day 전후는 BOFU 밀도 상향.
5. **슬롯 배치** — 날짜×채널 매트릭스에 주제 배치. 슬롯마다 기획 근거(어느 입력에서 왔나)와 실행 스킬(022/024/026/028/029) 매핑.
6. **주차별 플랜 + 체인 제안** — 주차 단위 제작 순서 정리 → 022 블로그 SEO 아티클(글감 실행) 또는 027 OSMU(1소스 다채널 확산).

```markdown
## 입력 근거 요약
- 캠페인: {n}건 ({주요 일정}) / 트렌드: {n}건 / VoC: {n}건 / 키워드: {n}건 {또는 [추정]}

## 월간 캘린더
| 날짜 | 채널 | 포맷 | 제목 가안 | 퍼널 | 기획 근거 | 실행 스킬 |
|---|---|---|---|---|---|---|
| 7/7(월) | IG | 캐러셀 | 21일 챌린지 인증 가이드 | TOFU | 캠페인(챌린지)+키워드 6위 | 026 |
| 7/8(화) | Blog | 아티클 | 모공 토너 고르는 기준 5가지 | MOFU | 키워드 1위(검색량 22,100) | 022 |
| 7/9(수) | Threads | 시리즈 | 펌프 불편하셨죠 | TOFU | VoC 1위 → 신제품 티저 | 029 |
| … | | | | | | |

## 채널 믹스 검산
IG {7} + Blog {4} + LinkedIn {4} + Threads {8} + NL {2} = {25} ✅ (캐파 일치)

## 주차별 발행 플랜

- …

## 다음 액션

```

## Anti-Patterns
- **근거 없는 글감 채우기** — "이런 것도 좋을 것 같아서"는 금지. 4입력 중 어디서 왔는지 못 대면 슬롯에서 뺀다.
- **캐파 초과 편성** — 슬롯이 많으면 좋아 보이지만 미발행 슬롯은 다음 달 신뢰를 깎는다. 확정 캐파 초과 금지.
- **TOFU 쏠림** — 인지 콘텐츠만 30건 깔면 전환이 없다. 퍼널 3단계 비율을 검산한다.
- **같은 주제를 전 채널에 복붙 배치** — 채널별 변형은 027의 몫이지만, 같은 날 같은 주제를 여러 채널에 나란히 놓는 배치 자체를 피한다.
- **발행까지 실행** — 캘린더는 Draft가 끝. 예약·발행은 승인 게이트를 가진 별도 절차로 넘긴다 (이중 안전망).

## 활용
- **금요일 기획 루틴**: 매주 금요일 이 스킬로 다음 주 슬롯을 확정 → 월요일엔 검토 15분만. "이번 주 뭐 올리지" 회의 2시간이 사라진다.
- **캠페인 시즌 운영**: 런칭 D-30 캘린더에 티저→오픈→증거→막판 슬롯을 역산 배치하고 BOFU 밀도를 D-Day에 수렴시킨다.
- **제작 파이프라인 연결**: 키워드 근거 슬롯은 022로, 영상 슬롯은 024→026으로, 앵커 1건은 027로 — 캘린더가 곧 스킬 실행 지시서가 된다.
