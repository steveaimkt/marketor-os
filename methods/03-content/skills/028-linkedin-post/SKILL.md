# 링크드인 포스트 빌더

> 주제 1개 → 부정 명령형 훅 + ❶❷❸ 번호 본문 + 숫자 대조 + 댓글 유도 CTA.
> 계승: `linkedin-post-writing`의 고성과 7패턴 (최고 성과 글 "맥미니 사지마세요" — 노출 39,357 · 댓글 261 분석 기반).

## Contract
- 첫 줄은 항상 부정 명령형("~하지 마세요") 또는 조건형("~하고 있다면") 훅이다. 질문형 단독 금지.
- 본문은 ❶❷❸ 번호 항목 3~5개로 구성되고, 각 항목은 굵은 소제목 + 2~3줄 설명이다.
- 충돌이 큰 숫자·비용 대조가 1개 이상 포함된다 ("119만원 → 월 1.3만원" 식).
- CTA는 댓글 유도형 1개 ("댓글로 '저요!' ✋")이며, 단순 링크 CTA로 끝나지 않는다.
- 산출물은 compliance-gate 통과 전에는 전달되지 않는다.

## Phases
1. **브랜드 컨텍스트 로드** — [brand/profile.md](../../../../brand/profile.md) 자동 참조 (한 줄 포지셔닝·경력 근거·금기 표현). 브랜드 정보 재질문 금지.
2. **훅 설계** — 부정 명령형/조건형 훅 3안 작성 → 최강 1안 채택, 나머지 2안은 대안으로 보존. 두 번째 줄에 숫자 대조를 예고.
3. **신뢰 형성 1문장** — 개인 경험·취약성 고백("저도 결제 직전까지 갔다가…") 또는 권위 경력 한 줄.
4. **❶❷❸ 본문** — 번호 항목 3~5개, 소제목은 본문과 빈 줄 없이 붙임. 산문 문단 금지. 본문 중 비용·숫자 대조 1개 이상 삽입, Before → After 3줄 비교 권장.
5. **CTA** — "가이드/혜택 받고 싶은 분, 댓글로 '저요!' 남겨주세요 ✋" 형식.
6. **모바일 포맷팅** — 블록(훅/도입/❶/❷/❸/Before→After/CTA) 사이에만 빈 줄 1칸, 블록 내부는 줄바꿈만, 긴 문장은 의미 단위 줄바꿈.
7. **자체 검증** — 훅 형식 / 숫자 대조 ≥1 / 번호 항목 3~5 / 경험 1문장 / 댓글 CTA / 캐주얼 슬랭 0 / 해시태그 기본 미사용.
8. **컴플라이언스 게이트** — [gates/compliance-gate.md](../../../../gates/compliance-gate.md) 3 STEP 실행, 게이트 결과 블록 부착 후에만 전달.

## Anti-Patterns
- **질문형 훅 단독** — "여러분은 ~하시나요?"로 시작하면 저성과 패턴. 반드시 부정 명령형/조건형.
- **산문 문단 나열** — 번호 구조 없는 긴 문단은 모바일에서 이탈. ❶❷❸로 쪼갠다.
- **링크 클릭 CTA** — "링크 확인하세요"는 댓글 0의 지름길. 댓글 유도형으로 교체.
- **해시태그 도배** — 오리지널 스타일은 해시태그 미사용이 기본.
- **블록마다 빈 줄 2칸 이상** — 모바일 읽기 리듬이 무너진다. 포맷팅 규칙 준수.

## 활용
- **강의·이벤트 모집**: 교육 런칭 소식을 "외주비 300만원 쓰고 있다면" 훅으로 전환, 댓글 신청자를 리드로 수집.
- **사례 콘텐츠화**: 고객 성과(Before→After 숫자)를 포스트로 변환해 신뢰 자산 축적.
- **주간 발행 루틴**: 021 콘텐츠 캘린더의 링크드인 슬롯 소재를 이 스킬로 일괄 작성 → 027로 타 채널 확장.
