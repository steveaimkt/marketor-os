# 콘텐츠 성과 회고

> 발행 콘텐츠 성과 분석(도달·참여·전환) → 승자 패턴 추출 → 다음 달 주제 추천 → 021 캘린더 피드백 루프.
> 계승: marketing-os `content-planning`의 성과 반영 루프(지난 기획·반응 조회 → 채널 성과 교차 → 다음 편성 반영)를 월간 회고 스킬로 분리.

## Contract
- 모든 성과는 채널별 정규화 후 비교한다 — 절대 수치 크로스 채널 비교 금지, 기준은 해당 채널 최근 3개월 중앙값 대비 배율.
- 지표는 3계층으로 분리 보고한다: 도달(노출·조회) / 참여(참여율·저장·댓글) / 전환(클릭·리드·매출 기여).
- 승자·패자 각 3건 이상을 뽑고, 패턴(주제×훅×포맷×발행 타이밍)을 콘텐츠 ID 근거와 함께 병기한다.
- 다음 달 주제 추천 5건에는 각각 "어떤 승자 패턴에서 왔는가" 근거 1줄이 붙는다.
- 데이터 미연결 지표는 [추정] 태그를 붙이며, 태그 없는 수치는 전부 실데이터다.

## Phases
1. **범위 확정** — 회고 기간·대상 채널·발행 목록 확인. 목록이 없으면 021 캘린더 산출물 또는 발행 기록에서 역구성.
2. **데이터 취합** — 소스 명시: YouTube Data MCP(조회·지속률·CTR), GA4/채널 인사이트(클릭·전환), 수기 시트. 미연결 채널은 사용자에게 수치 요청 ⏸ — 못 받으면 해당 지표 [추정] 모드로 표기하고 순위 산정에서 가중 하향.
3. **정규화·성적표** — 콘텐츠별 3계층 지표를 채널 중앙값 대비 배율로 환산 → 종합 성적표. 승자(중앙값 1.5배↑)·패자(0.5배↓) 추출.
4. **패턴 해부** — 승자·패자를 4축으로 분해: 주제(무엇을) × 훅(첫 3초/첫 줄) × 포맷(길이·형식) × 타이밍(요일·시간). 승자 공통항 2~3개 = 승자 패턴, 패자 공통항 = 회피 패턴. 표본 3건 미만 축은 "표본 부족" 명시 — 단정 금지.
5. **다음 달 주제 5건** — 승자 패턴 × 아직 안 다룬 소재 조합으로 생성. 각 건에 근거("{ID}가 중앙값 2.1배 — 같은 훅 유형") + 추천 채널·포맷 병기.
6. **021 피드백 루프** — 추천 5건을 021 콘텐츠 캘린더 빌더의 입력(주제 풀)으로 전달 제안. 회고→캘린더→발행→회고가 월간 사이클로 닫히는 구조를 명시.

## Output Format
```markdown
# 📊 콘텐츠 성과 회고: {기간}
**발행 {n}건 · 채널 {목록} · 데이터 소스: {실데이터/[추정] 비율}**

## 성적표 (채널 중앙값 대비 배율)
| ID | 콘텐츠 | 채널 | 도달 | 참여 | 전환 | 종합 | 판정 |
|---|---|---|---|---|---|---|---|
| C-12 | {제목} | YouTube | 2.4x | 1.8x | 2.1x | 2.1x | 🏆 승자 |
| C-07 | {제목} | Instagram | 0.4x | 0.3x | 0.2x | 0.3x | 📉 패자 |

## 패턴 분석
- **승자 패턴**: {훅 유형} + {포맷} + {요일} — 근거: C-12, C-03, C-18
- **회피 패턴**: {패자 공통항} — 근거: C-07, C-11
- 표본 부족 축: {있으면 명시}

## 다음 달 주제 추천 5건
| # | 주제 | 채널·포맷 | 근거 패턴 |
|---|---|---|---|

## 다음 액션
→ 021 콘텐츠 캘린더 빌더 (추천 5건을 주제 풀로 넣어 다음 달 편성 — 회고 루프 마감)
```

## Anti-Patterns
- **절대 수치 순위표** — 유튜브 조회수와 링크드인 노출을 한 줄에 세우면 항상 유튜브가 이긴다. 채널 중앙값 배율로만 비교.
- **도달만 보는 회고** — 조회수 승자가 전환 패자인 경우가 흔하다. 3계층을 분리해야 "잘 됐다"의 의미가 갈린다.
- **표본 1건으로 패턴 선언** — "화요일에 올린 게 터졌으니 화요일이 답"은 회고가 아니라 미신. 3건 미만은 표본 부족 표기.
- **추천 주제와 성과 데이터 단절** — 회고 따로 아이디어 따로면 이 스킬을 쓸 이유가 없다. 5건 전부 승자 패턴 근거 필수.
- **패자 콘텐츠 소재 폐기** — 죽은 건 소재가 아니라 패키징(훅·포맷)인 경우가 많다. 패자 소재 × 승자 훅 재조합을 우선 검토.

## 활용
- **월간 운영 사이클**: 매월 말 이 스킬 → 추천 5건을 021에 투입 → 다음 달 캘린더 확정. 회고가 기획의 입력이 되는 루프를 고정.
- **채널 손익 판단**: 3개월치 회고를 누적하면 채널별 전환 기여가 보인다 — 힘 뺄 채널·집중할 채널 결정 근거.
- **훅 라이브러리 갱신**: 승자 훅 패턴을 040 바이럴 훅 라이브러리의 "우리 채널 검증 패턴"으로 승격해 신규 콘텐츠에 우선 적용.
- **패자 재활용 실험**: 패자 소재 × 승자 훅 재조합 후보를 다음 달 추천 5건 중 1건에 포함 — 소재 폐기 전에 패키징 교체를 먼저 검증.

> 이 스킬은 내부 분석 산출물이므로 gate 대상이 아니다. 단, 회고 결과를 대외 콘텐츠(성과 자랑 포스트 등)로 쓸 때는 해당 제작 스킬의 게이트를 거친다.
