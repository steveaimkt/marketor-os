# OSMU 멀티채널 변환기

> 원본 1개 → 코어 추출 → 채널 문법별 재설계 4종+ → 게시 시차 전략까지 (전 변환본 게이트 일괄 통과).
> 계승: 멀티 콘텐츠 생성기 `/spread`의 6채널 변환 규칙·게시 전략 + 실습 3-2 content-publisher의 채널별 변형 규격·중복 검사.

## Contract
- 모든 변환본은 채널 문법 규격표를 따라 훅부터 재설계된다 — 원문 축약·복붙 변환 금지.
- 원본의 핵심 주장·수치는 채널이 바뀌어도 왜곡 없이 보존된다 (코어 메시지 카드와 전 변환본 대조).
- 변환본마다 게시 시차(D-Day 기준)와 채널 CTA가 지정된다.
- 최근 7일 내 같은 주제 발행 이력이 확인되면 중복 경고를 표기한다 (이력 미제공 시 검사 생략 명시).
- 전 변환본이 compliance-gate를 일괄 통과하기 전에는 전달되지 않는다. 발행·큐잉은 하지 않는다.

## Phases
1. **브랜드 컨텍스트 로드** — [brand/profile.md](../../../../brand/profile.md) + [brand/tone.md](../../../../brand/tone.md) 자동 참조 (채널별 변주 규칙 포함).
2. **코어 추출** — 원본에서 핵심 메시지 1개 · 뒷받침 주장 3개 · 수치·증거 목록 · 원본 CTA를 코어 메시지 카드로 고정. 이 카드가 전 채널 변환의 단일 기준.
3. **대상 채널 확인** ⏸ — 기본 4채널(링크드인·스레드·뉴스레터·블로그) + 확장(IG·X). 발행 이력이 있으면 7일 내 동일 주제 중복 검사.
4. **채널별 변환** — 각 채널의 전문 스킬 문법을 적용해 훅부터 재설계:
   | 채널 | 적용 문법 | 규격 |
   |---|---|---|
   | 링크드인 | 028 (부정 명령형 훅+❶❷❸+숫자 대조+댓글 CTA) | 모바일 포맷팅 |
   | 스레드 | 029 (분절 시리즈, 편당 500자) | 훅 포스트+연결 |
   | 뉴스레터 | 023 (두괄식 800자+제목 5종은 핵심 1종만) | CTA 1개 |
   | 블로그 | 022 (검색의도 구조+키워드 배치) | 2,000자+ |
   | IG(확장) | 캡션 1,500자 내+이모지+해시태그 | 캐러셀 구성안 |
   | X(확장) | 280자·핵심 1문장+수치 | 링크 1개 |
5. **게시 시차 전략** — 원본 채널 D-Day 기준 확산 배치 (예: D-Day 원본 → D+1 링크드인·뉴스레터 → D+2 스레드 → D+3 블로그). 채널 간 카니발리제이션 방지.
6. **일괄 게이트** — [gates/compliance-gate.md](../../../../gates/compliance-gate.md) 3 STEP을 전 변환본에 실행, 판정을 채널별로 표기. 1건이라도 ⛔면 해당 채널본 수정·재통과 후 전달.
7. **체인 제안** — 030 콘텐츠 성과 회고 (채널별 반응 비교 → 다음 원본 주제 선정).

## Anti-Patterns
- **길이만 줄인 복붙 변환** — 같은 문장을 자르기만 하면 전 채널에서 저성과. 채널마다 훅부터 다시 쓴다.
- **변환 중 수치 표류** — 채널을 옮기며 "2시간→15분"이 "90% 절감"으로 변형되는 것. 코어 카드와 전 변환본을 대조한다.
- **전 채널 동시 발행** — 같은 날 전 채널에 뿌리면 서로의 도달을 잡아먹는다. 시차 전략을 지킨다.
- **게이트 부분 통과 전달** — 4종 중 3종만 통과했는데 세트로 전달 금지. 일괄 통과가 조건이다.
- **발행·큐잉까지 실행** — 이 스킬은 변환본과 전략까지. Buffer·SNS 큐잉은 승인 게이트를 가진 별도 절차다.

## 활용
- **앵커 콘텐츠 주간 루틴**: 주 1회 앵커(유튜브·블로그) 발행 후 이 스킬 1회 실행 → 주간 4~6채널 발행량이 원본 1개 제작 코스트로 유지된다.
- **021 캘린더와 결합**: 캘린더의 "앵커+파생" 슬롯 구조를 이 스킬이 실물로 채운다 — 파생 슬롯의 기획 근거는 자동으로 "원본 D-Day".
- **런칭 증폭**: 신제품 발표문 1건을 4채널+IG·X 6종으로 변환해 런칭 주간 메시지 일관성과 도달을 동시에 잡는다.
