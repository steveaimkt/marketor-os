# 03-content — 콘텐츠 마케팅

## 이 카테고리가 해결하는 문제
콘텐츠 마케팅의 병목은 아이디어가 아니라 "꾸준한 생산량"이다. 캘린더로 기획하고, 블로그·뉴스레터·유튜브·숏폼을 제작하고, OSMU 변환기로 원본 1개를 N개 채널용으로 증식시켜 제작 단가를 낮춘다. 발행 후에는 성과 회고가 다음 달 주제를 다시 캘린더에 공급하는 순환 구조다.

## 추천 진입 경로
- **대표 커맨드: /콘텐츠프로덕션** — 체인 `021 → (022|024) → 027 → 030` (캘린더 → 블로그 또는 유튜브 스크립트 제작 → OSMU 멀티채널 변환 → 성과 회고)
- **처음이면 021(콘텐츠 캘린더 빌더)부터** — 한 달치 기획이 서면 022·024 제작 스킬이 주제를 골라 쓰기만 하면 된다.

## 다른 카테고리와의 연결
- **→ 08-crm**: 023(뉴스레터 제작 자동화) → 074(이메일 시퀀스 빌더) — 발행 포맷이 생애주기 메일 시나리오로 확장된다
- **← 04-social**: 040(바이럴 훅 라이브러리) → 026(쇼츠/릴스 대본 팩) — 검증된 훅 패턴이 숏폼 대본의 첫 3초를 공급한다
