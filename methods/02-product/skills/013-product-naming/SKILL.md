# 제품명·네이밍

> 네이밍 방향 확정 → 4유형×5 = 20안 생성 → 발음·표기·연상 3축 평가 → TOP 3 + 상표·도메인·핸들 실사 체크리스트.
> 이름은 "예쁜 것"이 아니라 "부를 수 있고, 검색되고, 등록 가능한 것"이다 — 생성과 실사를 한 번에 처리한다.

## Contract
- 후보는 정확히 20안 — 4유형(서술형/조어형/은유·연상형/한국어 어감형) × 각 5안. 유형이 비면 그 유형의 부적합 사유를 기록한다.
- 전 후보를 3축(발음 용이성 / 표기 안정성: 오타·외래어 표기 흔들림 / 연상 적합성: 가치제안 정렬) 각 5점으로 채점하고 근거를 병기한다.
- brand/tone.md의 네이밍 규칙·금기어와 대조해 위반 후보는 생성 단계에서 제외한다.
- TOP 3에는 상표(키프리스)·도메인·SNS 핸들·검색 오염도 체크리스트를 부착한다 — 체크리스트는 확인 절차 안내이지 등록 가능 판정이 아니며, 상표는 변리사 검토 대상임을 산출물에 명시한다.
- 후보의 실제 등록·사용 가능 여부를 단정하는 문장을 쓰지 않는다 ("가능합니다" 금지, "확인 필요" 표기).

## Phases
1. **브랜드 컨텍스트 로드** — [brand/profile.md](../../../../brand/profile.md) + brand/tone.md 자동 참조 (톤·기존 제품명 계열·금기어). 기존 라인업과의 계열 일관성 확인.
2. **방향 확정** — 012 가치제안문에서 이름이 실어야 할 핵심 연상 1~2개 추출 (예: "전문가급+합리적"). 신규 계열인지 기존 계열 확장인지 확인.
3. **20안 생성** — 유형별 5안: ①서술형(기능·용도 직설: 예 "가든컷") ②조어형(합성·축약·변형: 예 "볼트포스") ③은유·연상형(상징 이미지 차용: 예 "숲도끼") ④한국어 어감형(순우리말·의성어: 예 "싹둑"). 금기어·발음 곤란 후보는 생성 중 제외.
4. **3축 평가** — 발음(입으로 3번, 전화로 1번 테스트 관점)·표기(영/한 표기 흔들림, 오타 빈발 여부)·연상(핵심 연상 정렬) 각 5점 → 15점 만점 정렬.
5. **TOP 3 실사 체크리스트** — 후보별: 키프리스 상표 검색(동일·유사 호칭, 해당 니스 분류) / 도메인(.com·.co.kr·.kr 가용) / SNS 핸들(인스타그램·유튜브·네이버 스마트스토어명) / 검색 오염도(네이버·구글에서 동명 상품·부정 이슈 선점 여부). 각 항목은 확인 URL·방법 안내.
6. ⏸ **후보 확정** — 평가표+체크리스트를 보여주고 사용자가 실사 진행 후보를 확정한다 (실제 검색·조회는 사용자 실행 또는 승인 후 대행).
7. **산출 + 체인 제안** — 018 출시 체크리스트 & GTM 플랜 (확정명 기준 런칭 준비). 플랫폼 검색용 상품명 조합이 필요하면 053.

> 유형 정의 — 서술형: 기능·용도 직설 / 조어형: 합성·축약·변형 신조어 / 은유·연상형: 상징 이미지 차용 / 어감형: 순우리말·의성어

## Anti-Patterns
- **한 유형 몰빵** — 20안이 전부 조어형이면 발산이 아니라 변주다. 4유형 각 5안을 강제한다.
- **등록 가능 단정** — "이 이름은 상표 등록 가능합니다"는 이 스킬이 할 수 없는 말이다. 확인 절차와 변리사 검토 안내까지만.
- **뜻 좋은 발음 지옥** — 연상 점수만 보고 발음·표기 축을 무시하면 전화로 못 부르는 이름이 1위가 된다. 3축 동시 채점.
- **계열 무시 단발 네이밍** — 기존 라인업(예: ○○볼트 계열)과 무관한 이름은 브랜드 자산을 흩뜨린다. 2단계 계열 확인 생략 금지.
- **검색 오염 미확인** — 이미 유명 상품·부정 이슈가 선점한 이름은 SEO에서 영원히 밀린다. 체크리스트 4번을 빼먹지 않는다.

## 활용
- **신제품 런칭 체인**: 012 가치제안 확정 직후 실행해 TOP 3 실사 결과를 들고 018 GTM 킥오프에 들어간다.
- **라인 확장 네이밍**: 기존 계열 규칙(접두·접미 패턴)을 tone.md에 기록해두고 확장 제품마다 재실행 — 계열 일관성이 자동 유지된다.
- **리네이밍 판단**: 검색 오염도 체크만 단독 실행해 기존 제품명의 오염 수준을 진단하고 개명 여부를 판단한다.
