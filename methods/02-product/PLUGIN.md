# 02-product — 제품기획 & 포지셔닝

## 이 카테고리가 해결하는 문제
만들고 나서 팔릴 이유를 찾는 순서를 뒤집는다. 시장 갭에서 출발한 컨셉 → USP·가치제안 → 네이밍 → 가격 → GTM 플랜까지, 출시 전에 "왜 팔리는가"를 문서로 확정한다. 출시 후에는 A/B 테스트 설계와 개선 백로그 우선순위로 제품을 계속 다듬는다.

## 추천 진입 경로
- **대표 커맨드: /신제품런칭** — 체인 `011 → 012 → 017 → 018` (컨셉 브레인스토밍 → USP·가치제안 → 가격 전략 → GTM 플랜)
- **처음이면 011(신제품 컨셉 브레인스토밍)부터** — 아이디어가 이미 확정돼 있다면 012(USP·가치제안 정의)로 바로 진입해도 체인이 그대로 이어진다.

## 다른 카테고리와의 연결
- **→ 06-commerce**: 012(USP·가치제안 정의)·015(제품 스펙시트 → 셀링포인트) → 051(상세페이지 24섹션 빌더), 016(번들·프로모션 설계) → 057(프로모션 캘린더 & 행사 대응) — 기획 문서가 상세페이지·행사 딜의 원료가 된다
- **→ 05-ads**: 019(A/B 테스트 설계) → 047(광고 소재 A/B 판독기), 015(셀링포인트) → 043(메타 광고 소재 카피 팩) — 설계한 가설이 광고 실험으로 이어진다
