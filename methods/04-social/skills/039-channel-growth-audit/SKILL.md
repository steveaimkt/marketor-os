# 채널 성장 진단

> 계정 데이터를 퍼널로 분해해 **성장 병목 1곳**을 특정하고, 그 지점에만 집중한 우선 액션 3가지를 낸다.
> 계승: marketing-os `growth-audit` 에이전트 — 냉정한 사실 기반 진단(과장 금지), 강점/약점 분해, 저성과 콘텐츠 후킹 분석, 지난 진단 대비 변화 추적.

## Contract
- 병목은 유입→도달→참여→전환(팔로우/클릭) 퍼널의 특정 1구간으로 지목되고, 수치 근거가 병기된다.
- 우선 액션은 정확히 3가지, 전부 병목 구간에만 집중한다 ("전반적으로 개선" 금지).
- 진단은 사실 기반 — 데이터 없는 항목은 [추정] 태그를 붙이고, 칭찬을 위한 과장·위로성 진단을 하지 않는다.
- 저성과 콘텐츠 분석은 상위 3개 vs 하위 3개 대조로 훅·제목·썸네일 가설을 제시한다.
- 지난 진단 기록이 있으면 "지난 액션이 실제로 실행·개선됐는가"를 리포트 첫 항목으로 다룬다.

## Phases
1. **데이터 취합** — 소스 우선순위: ① 유튜브 = youtube-data MCP 자동 조회(채널 KPI + 영상별 성과, 다건은 병렬) ② 인스타·기타 = 인사이트 캡처/수치 수동 입력 요청 ③ 미제공 시 [추정] 모드 전환 고지 후 공개 지표(팔로워·게시 빈도·공개 반응)로만 진단. 기간 기본 최근 30일 + 직전 30일 비교.
2. **지난 진단 조회** — logs/ 또는 사용자 제공 이전 리포트가 있으면 지난 액션 3가지의 실행 여부·지표 변화를 먼저 확인 (변화 중심 진단).
3. **퍼널 분해 → 병목 특정** — 유입(노출·트래픽 소스)→도달(팔로워 도달률)→참여(ER·시청 지속)→전환(팔로우 전환·프로필 방문→팔로우) 구간별 수치화. 가장 큰 낙폭 1구간을 병목으로 지목. 참고 기준: 인스타 팔로우 전환(프로필 방문→팔로우) 3~5%, 릴스 비팔로워 도달 50%↑, 유튜브 노출 CTR 2~10% 구간 — 벤치마크 이탈 구간에 표시.
4. **콘텐츠 대조 분석** — 기간 내 상위 3 vs 하위 3 콘텐츠를 나란히 놓고 차이 가설: 훅(첫 3초/첫 줄), 제목·썸네일, 주제, 형식, 게시 시간. 가설엔 반드시 해당 콘텐츠 실례 인용.
5. **진단 정리** — 강점 2~3(뭐가 작동하나 — 유지 지시), 약점 2~3(어디서 새나), 병목 선언 1줄.
6. **우선 액션 3가지 → 체인** — 병목 구간 한정, 각 액션에 실행 방법·측정 지표·기대 변화 명시. 콘텐츠 기획 문제면 → 031, 훅 문제면 → 040 핸드오프.

## Output Format
```markdown
# 🔍 채널 성장 진단: {채널명} ({기간})
**병목**: {참여→전환} 구간 — {프로필 방문 4,200회 중 팔로우 1.1% (기준 3~5%)}

## 지난 진단 대비 (기록 있을 때)
{지난 액션 3개 실행 여부} → 좋아짐 / 정체 / 악화

## 퍼널 분해
| 구간 | 지표 | 이번 기간 | 직전 기간 | 판정 |
|---|---|---|---|---|
| 유입 | 노출 | 182K | 175K | ✅ |
| 전환 | 방문→팔로우 | 1.1% [추정] | 1.4% | 🔴 병목 |

## 강점 / 약점

- 상위 공통: {첫 줄에 숫자 결과 제시}
- 하위 공통: {훅 없이 제품 설명 시작} → 가설: {…}

## 🎯 우선 액션 3가지 (병목: {구간})
① {액션 — 방법 · 측정 지표 · 기대 변화}

## 다음 액션
→ 040 바이럴 훅 라이브러리 (하위 콘텐츠 훅 교체) · 031 인스타그램 피드 기획 (기획 리셋)
```

## Anti-Patterns
- **퍼널 전체에 액션 살포** — 병목 아닌 구간의 액션은 리소스 낭비. 3가지 전부 병목 구간에.
- **"꾸준히 하세요" 류 추상 액션** — 실행 방법·측정 지표 없는 액션 금지 (growth-audit 원칙: 추상 금지).
- **위로성 진단** — 약점을 흐리면 진단이 무의미. 사실 기반, 과장 없이.
- **단일 기간 스냅샷 판정** — 비교 기간 없이 "낮다/높다" 단정 금지. 직전 기간 또는 벤치마크 대비로만.
- **데이터 없이 확정 진단** — 인사이트 미제공인데 수치를 지어내는 것 금지. [추정] 태그 + 필요한 캡처 목록 요청.

## 활용
- **결로아 예시**: 인스타 12,400 팔로워 정체 진단 → 릴스 비팔로워 도달은 정상, 프로필 방문→팔로우 전환이 병목 → 프로필 리뉴얼 + 고정 게시물 3종 교체 액션.
- **월간 정기 감사**: 매월 초 같은 포맷으로 실행해 "지난 진단 대비" 추적 루프를 만든다 (개선 없는 액션은 폐기).
- **소셜그로스 커맨드 선두**: /소셜그로스 체인(039→031→040→035)의 진입 진단으로 사용 — 병목에 따라 후속 스킬이 갈린다.
