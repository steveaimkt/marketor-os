# 댓글·DM 응대 어시스턴트

> 댓글·DM 원문 → 3분류+감정점수 → 위기 키워드 스캔 → 브랜드 톤 응대 초안 (발송은 항상 수동).
> 계승: 디스코드 CS 응대 에이전트의 3분류(악성🔴/문의🟡/칭찬🟢)·감정점수 1~5·복합 판정 규칙·유형별 글자수·금지사항을 SNS 공개 문법으로 개조.

## Contract
- 모든 수신 건은 3분류(악성🔴/문의🟡/칭찬🟢) + 감정점수(1~5) + 긴급도 태그를 받는다. 복합 건은 악성>문의>칭찬 우선순위로 판정.
- 위기 키워드 검출 건은 응대 초안을 생성하지 않는다 — 즉시 에스컬레이션 카드로 분리해 089를 제안한다.
- 공개 댓글 답변은 "화난 1명"이 아니라 "지켜보는 100명"에게 쓴다 — 개별 분쟁 상세는 DM으로 전환 유도.
- 응대 초안은 brand/tone.md 톤·금기어를 따르고 글자수 제한(악성 200자/문의 300자/칭찬 150자)을 지킨다.
- 실제 발송은 사용자가 승인 후 수동으로 한다 — 이 스킬은 어떤 채널에도 자동 게시하지 않는다.

## Phases
1. **톤 로드** — [brand/profile.md](../../../../brand/profile.md) + brand/tone.md 자동 참조 (호칭·이모지 사용 수준·금기어).
2. **분류** — 건별 3분류 + 감정점수 + 긴급도. 판별: 악성=욕설·법적 언급·근거 없는 비방 / 문의=질문형·절차 확인·불편 설명 / 칭찬=긍정 표현·재구매 의사. 복합 규칙: 악성+문의→악성 우선, 문의+칭찬→문의 우선, 불명확→문의(안전 기본값).
3. **위기 키워드 스캔** — 사전 대조, 검출 시 해당 건 초안 중단 ⏸:

   | 계열 | 키워드 예 |
   |---|---|
   | 법적 | 고소, 소송, 변호사, 공정위, 소비자보호원, 신고 |
   | 언론·확산 | 기자, 제보, 방송, 커뮤니티에 올린다, 좌표 |
   | 안전·신체 | 부작용, 알레르기, 병원, 화상, 다쳤 |
   | 분쟁 반복 | 환불 3회 이상 언급, 같은 계정 반복 항의 |

   검출 건은 "위기 카드"(원문·키워드·권고 초동 대응)로 분리 → 089 위기 커뮤니케이션 플랜 체인.
4. **응대 초안 생성 (공개/DM 이원화)** —
   - **악성(공개)**: 감정 대응 금지, "불편을 드려 죄송합니다" 계열 시작 → 사실 확인 의지 → "DM으로 자세히 도와드릴게요" 전환. 보상 약속·변명·고객 주장 부정 금지.
   - **문의(공개 또는 DM)**: 정확한 정보만, 단계 안내는 1·2·3 번호, 모르는 건 지어내지 않고 확인 후 회신 약속.
   - **칭찬(공개)**: 고객이 언급한 포인트를 구체적으로 따와 감사 + 부드러운 재방문 유도. 과도한 리액션 금지.
   - 공통 금지: 경쟁사 언급, 내부 프로세스·시스템명 노출, 개인정보(주문번호 등)를 공개 댓글에 요구 — 개인정보 확인은 반드시 DM.
5. **컴플라이언스 게이트** — [gates/compliance-gate.md](../../../../gates/compliance-gate.md) 3 STEP (리뷰 답변도 대외 발행물), 결과 블록 부착.
6. ⏸ **발송 승인 + 패턴 기록** — 초안 일괄 검토 후 사용자가 채널에 수동 게시. 반복 문의 주제는 기록해 038 소셜 리스닝 테마와 대조 제안.

## Anti-Patterns
- **악성 댓글에 논리 반박** — 공개 댓글창에서 이기면 브랜드가 진다. 사과·사실 확인·DM 전환 3단만.
- **위기 건에 템플릿 답변** — "기자에게 제보하겠다"에 "불편을 드려 죄송합니다" 자동 답변은 기름 붓기. 위기 카드는 초안 생성 자체를 멈춘다.
- **공개 댓글에서 개인정보 요구** — "주문번호 남겨주세요"를 댓글로 하면 개인정보 노출 유도. 확인은 DM에서.
- **보상 선약속** — "환불해드리겠습니다"를 확인 전에 공개 답변으로 쓰면 선례가 된다. 고객센터·DM 절차 안내로 대체.
- **전 건 동일 말투** — 칭찬에 CS 상담 톤, 악성에 명랑 톤은 둘 다 사고. 분류별 구조·글자수를 따른다.

## 활용
- **아침 응대 루틴**: 밤새 쌓인 댓글·DM을 붙여넣으면 분류→위기 스캔→초안까지 일괄 처리, 사용자는 승인·게시만.
- **런칭·이벤트 직후**: 문의 폭증 구간에서 반복 질문 TOP 5를 뽑아 고정 댓글·FAQ 스토리로 선제 배포.
- **CS 자산화**: 승인된 응대 초안을 072 CS 응대 매뉴얼 빌더의 시드로 넘겨 채널 공식 매뉴얼로 승격.
