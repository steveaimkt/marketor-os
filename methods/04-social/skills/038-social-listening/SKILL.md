# 소셜 리스닝 리포트

> 브랜드 언급 수집 → 5분류 감성 태깅 → 주간 추이 리포트 → 부정 급증 시 089 조기 경보.
> 계승: 콘텐츠 댓글 분석기의 5분류(POSITIVE/NEGATIVE/QUESTION/SUGGESTION/NEUTRAL)·primary+secondary 복합 감정·반어법 탐지·influence_score(좋아요+답글×3)를 브랜드 언급 주간 리포트로 개조.
> **006과의 경계**: 038은 언급량·감성의 **흐름**을 매주 감시한다(넓고 얕게·시계열). 부정의 '왜'를 파려면 006 리뷰 마이닝(구매 이유·페인포인트 심층)으로 넘긴다.

## Contract
- 모든 언급에는 출처 링크·채널·수집일·감성 태그(primary, 필요 시 secondary)·confidence가 남는다 — 출처 없는 언급은 리포트에 올리지 않는다.
- 수집 실패·미연동 채널은 "미수집 채널"로 명시한다 — 언급을 지어내는 [추정] 모드는 이 스킬에 없다.
- 부정 비율이 전주 대비 2배 이상 또는 전체의 30%를 넘으면 리포트 최상단에 🚨 경보를 걸고 089를 즉시 제안한다.
- influence_score(좋아요+답글×3) 상위 20% 언급은 감성과 무관하게 "고영향" 섹션에 개별 노출한다.
- Notion 적재와 주간 자동 가동은 ⏸ 사용자 승인 후에만 실행한다 (mutating).

## Phases
1. **키워드 세트 확정** — 브랜드명 + 오타·띄어쓰기 변형 + 제품명 + 비교 문구("{브랜드} vs") + 대표 해시태그. 동음이의 오염 키워드는 제외어 등록.
2. **수집** — 소스 명시: 네이버 검색(블로그·카페·뉴스 MCP), YouTube 댓글(YouTube Data MCP), 커뮤니티·스레드(Firecrawl). 채널별 수집 건수·기간을 기록하고 URL 기준 중복 제거. 미연동 채널은 리포트에 "미수집" 표기.
3. **5분류 감성 태깅** — 건별: primary(POSITIVE/NEGATIVE/QUESTION/SUGGESTION/NEUTRAL) + secondary + confidence(0~1) + key_phrase(원문 발췌). 문맥 우선("좋은데 비싸요"→NEGATIVE primary), 반어법 신호(물결·과잉 칭찬+비판 병존) 시 confidence 0.5~0.7로 하향, confidence 0.5 미만은 "수동 검토" 섹션 분리. influence_score 계산.
4. **주간 지표 산출** — ① 언급량 WoW 증감 ② 감성 비율 추이(긍/부/중) ③ 테마 TOP 5(언급 군집화 — 각 테마에 대표 인용 1개) ④ 고영향 언급 목록 ⑤ QUESTION·SUGGESTION 군은 "대응 기회" 섹션으로 분리(032 응대·콘텐츠 소재 연결).
5. **경보 판정** — 부정 급증 조건 충족 시 🚨 블록(급증 테마·대표 원문·확산 채널) + 089 위기 커뮤니케이션 플랜 즉시 체인. 미충족 시 "정상 범위" 1줄.
6. ⏸ **Notion 적재** — 주간 리포트 1건을 Notion 아카이브 DB에 적재 (승인 후 실행). 4주 누적 시 월간 추이 비교 자동 포함.
7. ⏸ **상시 가동 제안** — 주 1회 자동 실행 스케줄 등록 여부 확인 (승인 없이는 1회성 실행으로 종료). 심층 분석이 필요한 부정 테마는 006으로 체인.

## Output Format
```markdown
# 📡 소셜 리스닝 주간 리포트: {브랜드} · {기간}
{🚨 부정 급증 경보 — 조건 충족 시에만 | ✅ 정상 범위}

## 요약
언급 {n}건 (WoW {±n}%) · 긍정 {n}% / 부정 {n}% / 중립·기타 {n}% · 수집: {채널 목록} (미수집: {…})

## 감성 추이 (4주)
| 주차 | 언급량 | 긍정 | 부정 | 비고 |
|---|---|---|---|---|

## 테마 TOP 5
| # | 테마 | 건수 | 감성 | 대표 인용 (출처 링크) |
|---|---|---|---|---|

## 고영향 언급 (influence 상위 20%)
| 채널 | 원문 요약 | 감성 | score | 링크 |
|---|---|---|---|---|

## 대응 기회 (QUESTION·SUGGESTION)
- {질문 테마} → 032 응대 · {제안 테마} → 콘텐츠 소재

## 수동 검토 (confidence < 0.5)
| 원문 | 잠정 분류 | 사유(반어법 가능 등) |

⏸ Notion 적재 승인 → {부정 테마 심층은 006 리뷰 마이닝} · {급증 시 089 위기 플랜}
```

## Anti-Patterns
- **출처 없는 요약** — "요즘 반응이 좋아요"는 리스닝이 아니다. 전 언급에 링크·채널·날짜.
- **미수집을 0건으로 보고** — 채널 미연동과 "언급 없음"은 다르다. 미수집 채널을 숨기면 추이가 왜곡된다.
- **키워드 오염 방치** — 동명 브랜드·일반명사 언급이 섞이면 감성 비율이 무의미해진다. 제외어를 매주 갱신.
- **부정 1건에 과잉 경보** — 경보는 비율·배수 조건으로만. influence 낮은 단발 악평에 089를 부르지 않는다.
- **여기서 구매 이유 심층 분석** — 부정의 '왜'를 파고들기 시작하면 주간 리포트가 늦어진다. 흐름은 038, 심층은 006 — 경계를 지킨다.

## 활용
- **주간 브랜드 관제**: 매주 월요일 자동 가동 → 10분 리뷰로 여론 이상 유무 확인, 4주 추이로 캠페인 전후 비교.
- **위기 조기 감지**: 부정 급증 경보 → 089 위기 플랜 가동까지의 리드타임을 "기사화 이후"에서 "커뮤니티 확산 초기"로 당긴다.
- **콘텐츠·CS 연료**: QUESTION 테마는 032 응대 템플릿·FAQ로, SUGGESTION 테마는 021 캘린더 주제 풀로 공급.
