# 바이럴 훅 라이브러리

> 검증된 8패턴(A~H)에서만 훅을 변형 생성하는 적용기. 즉흥 창작 금지.
> 계승: linkedin-post-writing 훅 공식 A~G(채널 불문으로 일반화 — 최고 성과 근거: "맥미니 사지마세요" 노출 39,357·댓글 261) + 유튜브 스크립트 에이전트의 슈퍼 후킹 3단(결과 선공개→놀라운 수치→최소 노력).

## Contract
- 모든 훅은 패턴 라이브러리(A~H)에서 변형 생성되며, 각 훅에 패턴 태그가 붙는다.
- 훅이 약속한 것을 본문이 증명하지 못하면(수치·근거 부재) 해당 훅은 폐기·재생성한다 — 낚시 차단.
- 채널별 훅 문법(쇼츠 0~3초 / 피드 첫 줄 / 텍스트 첫 1~2줄 / 롱폼 0~35초)을 구분 적용한다.
- 산출물은 10종 전부에 채널 적합 표시가 붙고, compliance-gate 통과 전에는 전달되지 않는다.

## Phases
1. **입력 확인** — 주제·채널·본문(또는 핵심 근거·수치). 본문 근거가 없으면 ⏸ "훅이 감당할 수치·결과물이 뭔가"를 먼저 확보.
2. **패턴 라이브러리 매칭** — 주제 유형별 우선 패턴 추천:

   | 패턴 | 공식 | 잘 맞는 주제 |
   |---|---|---|
   | A 조건형 주장 | "[결과] 원하면 [방법] 하세요" | 튜토리얼·노하우 |
   | B 문제-해결형 | "[문제]라면 [방법]으로 해결됩니다" | 페인포인트 소구 |
   | C 시간 약속형 | "[주제], 1분 안에 정리해드립니다" | 업데이트·뉴스 요약 |
   | D 공감 질문형 | "[대상] 잘 쓰고 계신가요?" + 공감 2~3줄 | 교육·커뮤니티 |
   | E 결과 선언형 | "[도구]로 [결과물] 만들어봤습니다" | 프로젝트·사례 |
   | F 스토리 유도형 | "가장 많이 받은 피드백은 하나였습니다" | 개선기·비하인드 |
   | G 뉴스/마일스톤형 | "[성과·사건] 소식을 전합니다" | 성과 공유 |
   | H 부정 명령형 | "[통념 행동] 하지 마세요" | 통념 반박 (최고 성과 패턴) |

3. **훅 10종 생성** — 우선 패턴 2~3개 × 변형으로 10종. 영상 채널은 유튜브 계승 규칙 적용: 결과물을 먼저 보여주고(0~5초) → 놀라운 수치 → "이걸 {n분} 만에" 최소 노력 강조. D 패턴은 질문 단독 금지 — 공감 후속 문장 필수.
4. **본문 감당 검증** — 훅별로 "본문의 어떤 근거가 이 훅을 증명하나" 1줄 매핑. 근거 없는 과장 훅(숫자 부풀림·"충격" 남발)은 폐기 후 재생성.
5. **채널 문법 적용** — 쇼츠/릴스: 0~3초 시각+음성 동시 훅, 자막 7단어 이내 / 피드·캡션: 첫 줄 잘림(~20자) 안에 핵심 / 텍스트(링크드인·스레드): 첫 1~2줄 + 숫자 대조 / 롱폼: 0~35초 3단 후킹 구조.
6. **컴플라이언스 게이트** — [gates/compliance-gate.md](../../../../gates/compliance-gate.md) 3 STEP (효능 과장·최상급 표현 근거 검사 포함), 결과 블록 부착.
7. **TOP 3 추천 → 체인** — 채널 적합도 기준 TOP 3 + 선택 훅으로 026 쇼츠 대본 또는 043 광고 카피 확장 제안.

```markdown

|---|---|---|---|---|
| 1 | H 부정 명령형 | "{통념} 하지 마세요" | {비용 대조 수치} | 텍스트 ◎ / 쇼츠 ○ |
| 2 | E 결과 선언형 | "{결과물} 만들어봤습니다" | {결과 스크린샷} | 피드 ◎ |
| … | | (10종) | | |

## TOP 3 추천

- 피드: 첫 줄 20자 "{잘림 안 핵심}"

─── 🛡 컴플라이언스 게이트 ───
판정: ✅ 통과 | ⚠️ 수정 후 통과 | ⛔ 차단
──────────────────────────

## 다음 액션
→ 026 쇼츠/릴스 대본 팩 (TOP 1 훅으로 대본 전개) · 043 메타 광고 소재 카피 팩 (훅을 광고 헤드라인으로)
```

## Anti-Patterns
- **패턴 밖 즉흥 창작** — "느낌 있는 첫 문장"은 재현 불가. 반드시 A~H 태그가 붙는 변형만.
- **본문이 못 갚는 훅** — "90% 절감"이라 해놓고 본문에 수치가 없으면 이탈+신뢰 손상. 근거 매핑 없는 훅은 폐기.
- **질문형 단독 훅** — "~하시나요?" 한 줄로 끝나는 훅은 검증된 저성과 패턴. D는 공감 후속 필수.
- **채널 문법 무시 이식** — 텍스트용 2줄 훅을 쇼츠 자막에 그대로 붙이면 3초를 놓친다. 채널별 변환 필수.
- **"충격"·"미쳤다" 남발** — 어그로 단어는 패턴이 아니다. 구체 수치·결과물이 훅의 재료.

## 활용
- **결로아 예시**: "수분 세럼 리뷰 1,200개 분석" 주제 → H "세럼 성분표만 보고 고르지 마세요" + E "리뷰 1,200개를 전부 읽어봤습니다" 등 10종 → 릴스 TOP 1을 026으로 전개.
- **저성과 콘텐츠 소생**: 039 진단에서 훅이 병목으로 나온 콘텐츠의 첫 3초/첫 줄만 교체해 재발행 실험.
- **광고 헤드라인 뱅크**: 오가닉에서 반응 확인된 훅 TOP 3를 043으로 넘겨 광고 소재 A/B 후보로 사용.
