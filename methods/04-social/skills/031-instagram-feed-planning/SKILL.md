# 인스타그램 피드 기획

> 브랜드 무드 → 3열 그리드 설계 → 유형 배합 → 30일치 캡션·해시태그까지 한 번에.
> 계승: marketing-os `content-planning`의 국면별 편성 로직(모집기/교육기 비중 조정) + `카드뉴스` 스킬의 디자인 토큰 추출·카피 게이트 감각(레퍼런스 품질 = 결과 품질).

## Contract
- 그리드는 9칸(3주기) 단위로 무드 일관성을 설계한다 — 프로필 3열 미리보기에서 배치 패턴(체커보드/세로 열/가로 줄)이 식별돼야 한다.
- 콘텐츠 유형 배합 비율이 수치로 명시되고, 30일 슬롯이 그 비율을 ±1칸 안에서 지킨다.
- 모든 캡션은 첫 줄 20자(더보기 잘림) 안에 훅을 넣고, 저장·공유 유도 장치를 정확히 1개 포함한다.
- 해시태그는 3계층(대형 10만+/중형 1만~10만/니치 1만 미만) 배합으로 5~15개, 게시물 간 세트 복붙 금지.
- 산출물은 compliance-gate 통과 전에는 전달되지 않는다.

## Phases
1. **브랜드·목적 로드** — [brand/profile.md](../../../../brand/profile.md) + brand/tone.md 자동 참조. 계정 목적(팔로워 성장/판매 전환/커뮤니티)과 보유 사진 자산 수준(촬영 가능/보유 컷만/생성 이미지) 확인.
2. ⏸ **무드·그리드 패턴 확정** — 팔레트 3색 + 촬영 무드 키워드 3개 + 배치 패턴 1개를 제안하고 승인받는다:
   - **체커보드**: 비주얼 컷 ↔ 텍스트 카드 교차 — 정보성 계정
   - **세로 열 고정**: 가운데 열=인용/카드뉴스 고정 — 브랜드 메시지 계정
   - **가로 줄 테마**: 3칸씩 한 캠페인/시리즈 — 제품 런칭 계정
3. **유형 배합 설계** — 목적별 기본 배합(조정 가능): 가치·교육 40% / 제품·오퍼 20% / 신뢰(후기·비하인드) 20% / 참여 유도(질문·이벤트) 10% / 릴스 10%. 판매 목적이면 제품·오퍼 30%까지, 오퍼 게시물 연속 배치 금지.
4. **30일 슬롯 배치** — 날짜×유형×그리드 위치를 배치 패턴에 맞춰 편성. 각 슬롯에 소재 1줄(무엇을 찍/만들지)과 3열 미리보기상 위치(좌/중/우) 표기.
5. **캡션 30건 작성** — 구조 고정: 첫 줄 훅(20자) → 본문 2~4줄(tone.md 어휘) → CTA 1개 → 저장 유도 1개("나중에 볼 거면 저장" 류는 유형별로 문구 변주). 오퍼 게시물은 가격·혜택 표기 정확성 우선.
6. **해시태그 세트** — 게시물별 3계층 배합(대형 2~3 + 중형 4~6 + 니치 3~5). 브랜드 고정 태그 1~2개 외에는 게시물 소재에 맞게 개별 구성.
7. **컴플라이언스 게이트** — [gates/compliance-gate.md](../../../../gates/compliance-gate.md) 3 STEP (오퍼·효능 표현, 대가성 표기 집중), 결과 블록 부착.
8. **체인 제안** — 040 바이럴 훅 라이브러리 (첫 줄 훅이 약한 슬롯의 훅 10종 재생성).

## Anti-Patterns
- **낱장 기획** — 게시물 하나하나는 예쁜데 프로필 그리드가 뒤죽박죽이면 팔로우 전환이 죽는다. 9칸 단위로 먼저 본다.
- **오퍼 도배** — 판매 게시물 연속 3칸이면 피드가 전단지가 된다. 배합 비율과 연속 배치 금지 준수.
- **첫 줄에 인사말** — "안녕하세요 {브랜드}입니다"는 더보기 잘림 20자를 버리는 짓. 첫 줄은 무조건 훅.
- **해시태그 세트 복붙** — 30일 내내 같은 태그 세트는 도달 하락 신호. 계층 배합은 유지하되 태그는 소재별로.
- **사진 자산 무시 기획** — 촬영 여력이 없는데 매일 신규 촬영 컷을 요구하는 캘린더는 실행 불가. 자산 수준을 Phase 1에서 확인한다.

## 활용
- **신규 계정 세팅**: 첫 9칸을 브랜드 소개 미니 시리즈로 설계 — 프로필 방문자가 3초 안에 "뭐 하는 계정인지" 읽히게.
- **제품 런칭 월**: 가로 줄 테마 패턴으로 런칭 주간 3칸을 티저→공개→후기로 배치, 오퍼 비중을 30%로 상향.
- **월간 루프**: 이 스킬 30일치 → 발행 → 030 성과 회고에서 저장률 상위 유형 확인 → 다음 달 배합 비율 조정.
