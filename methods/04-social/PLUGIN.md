# 04-social — SNS 운영 & 커뮤니티

## 이 카테고리가 해결하는 문제
계정은 열었는데 "뭘 올릴지, 왜 안 크는지"를 모르는 상태를 해결한다. 채널 성장 진단으로 병목을 짚고, 피드 기획·바이럴 훅·이벤트로 성장 레버를 당기며, 댓글 응대·소셜 리스닝·커뮤니티 운영으로 팬을 관리한다. 인플루언서와 UGC까지 외부 크리에이터 자원도 이 카테고리가 담당한다.

## 추천 진입 경로
- **대표 커맨드: /소셜그로스** — 체인 `039 → 031 → 040 → 035` (채널 성장 진단 → 피드 기획 → 바이럴 훅 적용 → 이벤트 설계)
- **처음이면 039(채널 성장 진단)부터** — 병목이 무엇인지 먼저 확정해야 031·040·035가 정확한 처방이 된다.

## 다른 카테고리와의 연결
- **→ 09-brand-sales**: 032(댓글·DM 응대)·038(소셜 리스닝 리포트) → 089(위기 커뮤니케이션 플랜) — 위기 키워드 감지가 대응 플랜 가동으로 직결된다
- **→ 03-content**: 037(UGC 수집·활용 시스템) → 027(OSMU 멀티채널 변환기), 040(바이럴 훅) → 026(쇼츠/릴스 대본 팩) — 고객 콘텐츠와 훅 패턴이 콘텐츠 공장의 원료가 된다
