# 076 멤버십·등급 제도 설계

> 065 RFM 세그먼트의 매출 기여 분포로 등급 경계를 정하고, 혜택 원가를 반영한 마진 시뮬레이션으로 "퍼주다 망하는 멤버십"을 설계 단계에서 차단한다.
> 계승: 065 고객 세그먼트 RFM 분석의 세그먼트 정의표(M 분위·매출 기여) → 등급 경계 근거 / 016 번들·프로모션 설계의 할인 후 실마진 시뮬레이션 공식 재사용.

## Contract
1. 등급 경계는 실제 구매 분포 데이터(065 세그먼트 리포트의 M 분위·매출 기여)로 정한다 — "6개월 50만원" 같은 감 기준을 업종 불문 적용하지 않는다. 데이터 없으면 [추정] 태그 필수.
2. 모든 혜택은 원가로 환산되어 등급별 실마진 시뮬레이션에 반영된다 (실마진 = 등급 매출 × 마진율 − 혜택 원가 합계). 원가 미산정 혜택은 구조표에 넣지 않는다.
3. 등급별 혜택 원가율이 해당 등급 마진을 초과하는 설계는 전달하지 않는다 — 초과 시 혜택 축소안을 함께 제시한다.
4. 승급 규칙과 함께 강등·유지 규칙(산정 주기·유예)을 반드시 설계한다 — 승급만 있는 제도는 미완성이다.
5. 같은 구매 분포를 다시 넣으면 같은 등급 경계가 나온다 (경계 산출 기준을 산출물에 명시).

## Phases
1. **브랜드·데이터 로드** — brand/profile.md(마진율·AOV·구매주기) 자동 참조. 065 세그먼트 리포트가 있으면 세그먼트 정의표·매출 기여를 등급 재료로 로드. 없으면 065 선행을 권고하고 ⏸, 그래도 진행 시 구매 분포 직접 입력 + [추정] 모드.
2. **등급 구조 설계** — 3~4등급 (5등급 이상 금지). 경계는 산정 기간(최근 6~12개월) 구매액 분위로: 예) VIP = 상위 5% / GOLD = 상위 20% / 일반. 065 세그먼트와 매핑 (챔피언→VIP, 충성→GOLD)해 등급별 예상 인원·매출 기여를 산출.
3. **혜택 설계** — 등급별 차등표 (할인율·적립률·무료배송·전용 혜택·선공개). 상위 등급일수록 "비용형 혜택(할인)"보다 "경험형 혜택(선공개·전용)"의 비중을 높이는 원칙 적용.
4. **마진 시뮬레이션** — 016 공식 재사용, 등급별로:
   `혜택 원가 = 예상 매출 × (할인율 + 적립률 × 사용률) + 무료배송 원가 × 주문 수 + 전용 혜택 원가`
   `실마진 = 등급 매출 × 기존 마진율 − 혜택 원가` → 등급별 혜택 원가율(%)과 손익 표. 리텐션 개선 가정(보수/기본/낙관 3안)으로 회수 기간 제시.
5. **승급·강등·유지 규칙** — 산정 주기(분기/반기), 강등 유예(1주기), 등급 통보 시점·채널. 등급 산정 로직은 한 문단으로 명문화 (CS 분쟁 대비).
6. ⏸ **구조 확정** — 시뮬레이션 3안 중 사용자가 선택·조정 → 최종 구조표 인도.
7. **체인 제안** — 065 고객 세그먼트 RFM 분석 (분기마다 재실행해 등급 인원 이동·경계 리밸런싱).

## Anti-Patterns
- **감으로 등급 경계** — "왠지 50만원이 VIP"는 인원이 0.5%거나 30%로 튄다. 반드시 분포 분위에서 도출.
- **혜택 원가 미계산** — 적립률 5%가 마진 15% 업종에서 어떤 의미인지 계산 없이 확정하는 것. 적립은 사용률까지 곱해야 진짜 원가다.
- **승급만 설계** — 강등 규칙이 없으면 2년 뒤 전원 VIP가 된다. 강등·유예·통보까지가 한 세트.
- **등급 인플레이션** — 5~7등급으로 쪼개면 등급 간 혜택 차가 미미해져 승급 동기가 죽는다. 3~4등급 상한.
- **전 등급 할인 중심 혜택** — 상위 등급까지 할인으로만 채우면 멤버십이 상시 세일이 된다. 상위는 경험형 혜택 비중을 높인다.

## 활용
- **D2C 멤버십 런칭**: 065 리포트의 챔피언·충성 세그먼트 규모로 VIP·GOLD 경계를 잡고, 시뮬레이션 기본안으로 3개월 파일럿 후 리밸런싱.
- **기존 등급제 수익성 점검**: 이미 운영 중인 등급제의 혜택 원가율을 ③표로 재계산 — 적자 등급을 찾아 혜택 재배치.
- **VIP 전용 캠페인 연계**: 등급 구조표를 074 이메일 시퀀스(등급별 오퍼 분리)·075 알림톡(승급 축하·강등 예고)의 타겟 기준으로 전달.
