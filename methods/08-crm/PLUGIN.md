# 08-crm — CRM & 고객관리

## 이 카테고리가 해결하는 문제
신규 유입에만 돈을 쓰고 기존 고객을 방치하는 구조를 뒤집는다. 고객 여정을 맵핑하고 이메일·알림톡 시퀀스, 멤버십, 윈백 캠페인으로 생애주기 전체에 재구매 장치를 심는다. CS 쪽에서는 챗봇·매뉴얼·VOC 분류로 응대를 시스템화하고, 고객의 소리를 제품 개선 백로그까지 연결한다.

## 추천 진입 경로
- **대표 커맨드: /리텐션캠페인** — 체인 `065 → 073 → 074 → 077` (RFM 세그먼트 분석 → 고객 여정 맵핑 → 이메일 시퀀스 → 이탈 방지 윈백)
- **첫 스킬은 065(고객 세그먼트 RFM 분석, 07-analytics 소속)** — 고객이 등급으로 나뉘어야 073 이후의 시퀀스가 "누구에게"를 갖는다. 이 카테고리 안에서만 시작하려면 073(고객 여정 맵핑)부터.

## 다른 카테고리와의 연결
- **→ 02-product**: 078(VOC 취합·분류) → 020(제품 개선 백로그 우선순위), 080(고객 인터뷰) → 012(USP·가치제안 정의) — 고객의 소리가 제품 설계로 순환한다
- **→ 06-commerce**: 079(리뷰·후기 요청 자동화) → 054(리뷰 관리 시스템) — 요청으로 모은 리뷰가 스토어 리뷰 운영 루프에 쌓인다
