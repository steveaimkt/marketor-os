# 069 예측 시뮬레이션

> 광고비·전환율·객단가 3변수를 흔들어 **보수/중립/공격 3안 매출 시나리오**를 만든다. 점쟁이가 아니라 계산기 — 모든 예측치에 전제와 오차 범위가 붙는다.
> 계승: 056 쿠팡 운영의 검증된 가중 예측 공식(**최근 7일 가중 50% + 전체 평균 50%**)을 기준선 산출에 재사용. 원본 재고 분석 에이전트의 품절 예측 로직을 매출 시나리오로 일반화.

## Contract
- 기준선(현상 유지) 예측은 고정 공식을 쓴다: **일평균 = 최근 7일 가중 평균 × 50% + 기간 전체 평균 × 50%** → 기간 환산. 공식과 입력값을 산출물에 명시한다 (재현성).
- 시나리오는 보수/중립/공격 3안이며, 각 안의 변수 가정(광고비·전환율·객단가)이 표로 명시된다 — **전제 없는 예측치는 내보내지 않는다**.
- 모든 예측치에 `[추정]` 태그와 오차 범위(과거 데이터의 주간 변동폭 기반 ±%)를 붙인다 — "매출 오릅니다"류 보장 표현 금지 (profile.md §6).
- 민감도 분석으로 매출에 가장 영향 큰 변수 1개를 특정한다 — 시나리오 나열로 끝내지 않는다.
- 체감 규칙을 명시한다: 광고비 증액 시나리오에는 효율 체감(CPC 상승·전환율 유지 가정의 한계)을 반드시 전제에 기록한다.

## Phases
1. **기준 로드** — [brand/profile.md](../../../../brand/profile.md) §4 AOV·마진율·목표 ROAS 자동 참조. 미기록이면 데이터에서 추정 + `[추정]` 태그 (재질문 금지, 갱신 권고만).
2. **데이터 확보·검증** — 일별 매출·광고비·전환 데이터 로드 (CSV·시트 폴백). 최소 4주 미만이면 "예측 신뢰 한계" 경고를 붙이고 오차 범위를 2배로 확대. 프로모션·행사 일자는 이상치로 태그해 기준선 계산에서 보정.
3. **기준선 산출** — 가중 공식(최근 7일 50% + 전체 평균 50%)으로 일평균 → 예측 기간 환산. 과거 주간 변동폭(±σ)으로 오차 범위 산출.
4. ⏸ **변수 범위 합의** — 3변수 조정 폭 기본안을 제시하고 사용자가 확정한다: 보수(광고비 -20%·전환율 -10%·객단가 유지) / 중립(현행 유지) / 공격(광고비 +50%·전환율 +10%·객단가 +5%). 사용자의 실제 계획(행사·신제품)이 있으면 그 값으로 교체.
5. **시나리오 계산** — 구조식: 예상 매출 = (광고비 ÷ CPC × 전환율 × 객단가) + 오가닉 기준선. 3안 각각 매출·광고비·ROAS·마진 기여를 산출하고 손익분기 여부를 판정. 공격안에는 CPC 상승 가정(+10~20%)을 기본 반영한다.
6. **민감도 분석** — 변수 1개씩 ±10% 단독 조정 시 Δ매출을 비교 → 최대 영향 변수 특정 + "이 변수부터 개선하라" 해석 1줄.
7. **전제 명시·체인** — 전제 목록(효율 체감·시즌성 미반영·데이터 기간)을 산출물 하단에 고정 배치 → 095 마케팅 예산 플래너로 핸드오프 (선택 시나리오의 광고비를 월별 예산으로 전개).

## Output Format
```

### ② 시나리오 3안 ⏸

|---|---|---|---|---|---|---|

### ④ 전제 목록 (이 전제가 깨지면 예측도 깨진다)
```

## Anti-Patterns
- **전제 없는 단정 수치** — "다음 달 매출 5,200만원"만 내보내는 것. 전제·오차 범위 없는 예측은 게이트 이전에 스킬 위반.
- **선형 확장 환상** — 광고비 2배 = 매출 2배로 계산하는 것. 증액 시나리오에는 CPC 상승·오디언스 포화 체감을 전제에 박는다.
- **행사 주간 포함 기준선** — 프로모션 스파이크를 평시 평균에 섞으면 기준선이 부풀어 모든 시나리오가 낙관 오염된다.
- **시나리오 나열로 종료** — 3안 표만 주면 사용자는 "그래서 뭘 하지"가 된다. 민감도 1위 변수와 권장 안 1개를 지목한다.
- **보장 화법** — "공격안으로 가면 매출이 오릅니다" 금지. "이 전제가 유지되면 이 범위가 나온다"가 이 스킬의 문법이다 (profile.md §6).

## 활용
- **예산 증액 품의**: 공격안의 매출·ROAS 범위 + 전제 목록을 그대로 품의 근거로 — "얼마 늘리면 얼마 나오나"에 범위로 답한다.
- **분기 목표 현실성 검증**: 목표 매출을 시나리오와 대조 — 공격안으로도 안 닿으면 목표가 아니라 소원이라는 걸 회의 전에 안다.
- **061 연계 정례화**: 061 판매 데이터 분석의 요인 분해 결과(수량/단가/믹스)를 변수 조정 폭의 근거로 쓰면 가정이 감이 아니라 데이터가 된다.
- **사후 검증 루프**: 예측 기간 종료 후 실적과 대조해 오차율을 기록 — 다음 실행에서 오차 범위 산정이 실측 기반으로 좁혀진다.

> 부록 — 시나리오 구조식 분해 (검산용)

```
예상 매출 = 유료 기여 + 오가닉 기준선
  유료 기여   = 광고비 ÷ CPC × 전환율 × 객단가
  오가닉 기준선 = 가중 일평균(유료 제외분) × 기간 일수
검산: 중립안 ≈ 기준선 (오차 ±5% 초과 시 모델 재점검)
```
