# 07-analytics — 데이터 분석 & 리포팅

## 이 카테고리가 해결하는 문제
데이터는 쌓이는데 "그래서 뭘 해야 하는지"가 안 나오는 상태를 해결한다. 지저분한 로우데이터를 정리해 판매·GA4·코호트·RFM 분석으로 성장 동인과 병목을 짚고, KPI 트리와 예측 시뮬레이션으로 다음 결정을 뒷받침한다. 산출물은 대시보드와 월간 리포트로 의사결정자가 바로 읽는 형태가 된다.

## 추천 진입 경로
- **대표 커맨드: /월간리포트** — 체인 `068 → 061 → 067 → 066` (데이터 클리닝 → 판매 데이터 분석 → 월간 리포트 자동화 → KPI 트리 점검)
- **처음이면 068(데이터 클리닝 & 전처리)부터** — 로우데이터가 표준 형태로 정리돼야 061 이후의 모든 분석이 신뢰할 수 있는 숫자 위에서 돈다.

## 다른 카테고리와의 연결
- **→ 08-crm**: 065(고객 세그먼트 RFM 분석) → 073(고객 여정 맵핑)·074(이메일 시퀀스 빌더), 064(코호트·리텐션 분석) → 077(이탈 방지 윈백 캠페인) — 세그먼트·리텐션 진단이 캠페인 설계의 근거가 된다
- **→ 10-ops**: 067(월간 리포트)·069(예측 시뮬레이션) → 095(마케팅 예산 플래너) — 결산과 시나리오가 다음 달 예산 배분으로 직결된다
