# 광고 카피 변형 스킬

## 10패턴 (Claude는 반드시 10개 모두 생성)

| # | 패턴 | 공식 | 예시 |
|---|---|---|---|
| 1 | 문제 제기 | "{문제 상황}, {해결책}" | "새벽 모공 푸석함, 21일에 끝" |
| 2 | 숫자 약속 | "{기간} 후 {정량 결과}" | "21일 후 모공 -38%" |
| 3 | 반전 | "{통념} 사실 {반박}" | "스킨케어 7단계, 사실 틀렸어요" |
| 4 | 공감 | "{타겟} 모두 {공감}" | "20대 모두 가진 이 고민" |
| 5 | 이름 호출 | "{이름}님 {호기심}" | "민지님이 놓치고 있는 것" |
| 6 | 마감 압박 | "D-{N} {혜택}" | "마감 D-3 토너 1+1" |
| 7 | 소셜 프루프 | "{N}명 인증" | "구독자 1,247명이 인증" |
| 8 | 호기심 GAP | "왜 다들 {행동}했나" | "왜 다들 이 토너로 갈아탔나" |
| 9 | 데이터 헤드라인 | "{출처} {수치}" | "서울대 임상 결과: 38% 축소" |
| 10 | 명령형 CTA | "{명령} {혜택}" | "오늘만 -30%, 지금 받기" |

## 입력
```yaml
core_message: "토너 출시"
character_limits:
  headline: 25
  body: 90
brand_voice: "(자동 로드)"
```

## 출력
```yaml
variants:
  - pattern: "문제 제기"
    headline: "..."
    body: "..."
    cta: "..."
  - pattern: "숫자 약속"
    ...
  # 총 10개
```

## 검증
- 10개 모두 패턴 1:1 매칭
- 글자 수 제한 위반 0
- 금기어 0
- 본 brand-voice 어휘 점수 ≥ 3
