# 마케팅팀 구축하자 · 조직 운영 진단 → 사람+AI 팀 재구성

> **무엇을 하나**: 마케팅 조직의 **운영·관리 문제를 진단하고 해결**하면서, 실제 직원(사람)과
> marketing-os 자산(에이전트 71·방법론 100스킬·MCP·하네스)을 하나의 **하이브리드 팀**으로 조립한다.
> AI 봇만 편성하는 게 아니라 — 사람이 무엇에 집중하고, AI가 무엇을 덜어내는지 재설계한다.
> **결과물**: 조직 운영 진단서 + 하이브리드 팀 제안서 + `brand/team.md`(직원+AI+운영)·`org-map.md`·`roster.md`·`marketer.md`.

## 실행 원칙
- **자동 실행 금지**. 각 단계 질문 → 답 → 다음.
- 조직도(직급)가 아니라 **실제 반복 업무 + 운영 문제**로 팀을 짠다.
- **사람과 AI를 함께 관리** — 직원 명부·부하·책임을 등록하고, AI는 그 위에서 보조/대체를 매핑한다.
- 진단은 근거를 댄다("좋아 보임" 금지 · 업무·시간·부하 수치로).
- msk 없이 자립 (방법론 정본은 `methods/` 참조).

## STEP 1 · 현황 파악 → org-map + team.md
말문:
> "마케팅팀을 구축해드립니다. 봇만 붙이는 게 아니라, 지금 조직의 운영 문제를 먼저 풀고
> 사람과 AI가 함께 일하는 팀으로 다시 짭니다. 몇 가지 여쭐게요."

### ⓪ 자료로 먼저 분석 (선택 · 인터뷰 시간 단축)
> "질문 전에, 회사를 빠르게 파악할 **자료나 링크**가 있으면 주세요. 제가 먼저 읽고 초안을 채운 뒤 확인만 받을게요."
- **회사 홈페이지·랜딩·인스타/유튜브 링크** → `firecrawl_scrape`/`WebFetch`로 업종·제품·톤·채널 자동 파악
- **경쟁사 링크** → 스크랩·비교 (STEP 2 진단·벤치마크 근거로 적재)
- **기존 마케팅 자료·조직도·업무 리스트 (파일/캡처 이미지)** → Read(이미지 포함)로 직원·업무 초안 추출
→ 분석 결과를 아래 항목에 **미리 채우고**, 빈 곳·확인만 질문한다. (자료 없으면 바로 질문)

### 질문 (객관식 + 주관식 혼합 · 답 받고 다음)
1. **회사·업종**
   - (객관식) 업종: ① 화장품/뷰티 ② 식품/건기식 ③ 패션 ④ 금융 ⑤ 교육 ⑥ IT/SaaS ⑦ 커머스/유통 ⑧ 기타 *(→ 규제 게이트 결정)*
   - (주관식) 한 줄로 뭘 파는 회사인지
2. **실제 직원 명부** ★ (주관식 · 표로 · 자료 있으면 초안 확인)
   - 각자: 이름/호칭 · 역할 · 담당 업무 · **부하**(① 여유 ② 적정 ③ 과부하) · 강점
3. **반복 업무** (객관식 다중선택 + 주관식)
   - (다중선택) □뉴스레터 □SNS콘텐츠 □상세페이지 □광고운영 □경쟁분석 □SEO □CS/리뷰 □이메일CRM □정기리포트 □기타
   - (주관식) 체크한 각 업무: 담당·빈도·소요시간·쓰는 도구
4. **운영 구조**
   - (객관식) 승인 성향: ① 건건이 확인 ② 방향만 맞으면 위임 ③ 발행·지출만 확인
   - (주관식) 승인권자 · 보고선 · **어디서 막히는지**
5. **성과·목표** (주관식)
   - 핵심 KPI · 잘 되는 것 / 막힌 것

→ `brand/team.md`(직원 명부)·`brand/org-map.md`(§1~2)에 적재. 자료 분석 출처는 `[분석]` 태그, 미확인은 `[확인필요]`.

## STEP 2 · 조직 운영 진단 (★ 핵심 · 문제 해결의 출발)
STEP 1로 **운영 진단서**를 만든다. 사람 조직의 실제 문제를 짚는다:
1. **과부하·병목** — 특정 직원에 일이 몰림 / 한 사람 대기로 전체가 멈춤
2. **커버 갭** — 해야 하는데 아무도 안 하는 업무 (예: 경쟁 모니터링·SEO)
3. **수작업 낭비** — 자동화 가능한데 사람이 반복 (시간 계산)
4. **책임·검수 공백** — 발행 승인 없음 / 규제 검수 없음 / 성과 측정 안 됨
5. **강점** — 이미 잘 하는 것 (여기엔 AI를 보조로만)

→ 1화면 진단서:
```
📋 {회사} 마케팅 조직 운영 진단
👥 인력: {N}명 · 과부하 {누구}({무슨 업무})
⚠️ 커버 갭: {아무도 안 하는 업무}
🔧 수작업 낭비: {업무}({주 N시간})
🚨 책임 공백: {승인·검수·측정 부재}
✅ 강점: {잘 하는 것}
```
⏸ "이 진단 맞나요? 빠진 문제 있나요?" → 보정.

## STEP 3 · 하이브리드 팀 구성 (사람 재정의 + AI 매핑) → team.md + roster.md
진단을 근거로 **사람과 AI를 함께** 재편한다.

**A. 사람 역할 재정의** — 각 직원이 무엇에 **집중**하고 무엇을 **AI에 넘기는지**:
```
{직원명} · {역할} → 집중: {고부가 업무} / AI 위임: {반복 업무} → 부하 {과부하→적정}
```

**B. AI 에이전트 매핑** — 업무·갭에 대응하는 에이전트만 켠다:
- 항상: 트루먼·gate-auditor·quality-reviewer·perspective-reviewer·skill-builder
- 부서장: 업무 카테고리 있는 것만 (광고 안 하면 ads-lead ⬜)
- 실무: 업무·갭에 매핑된 것만 · 각 에이전트에 **어느 직원을 보조하는지** 명시

**C. 가감**:
- ➕ **추가** — 기존 71명·100스킬·MCP로 안 되는 업무면 skill-builder로 신규 제작 제안
- ➖ **제외** — 해당 업무 없는 에이전트·안 쓰는 MCP는 끈다
- **검증 딱지** — [검증됨]/[템플릿](광고 6개 등 미검증)

→ **하이브리드 팀 제안서** 1화면:
```
🏗 {회사} 마케팅팀 (사람 {N}명 + AI {M}명)
- 사람: {직원별 집중 업무}
- AI 코어 5 + 부서장 {a} + 실무 {b}
- 사람↔AI 매핑: {누가 무슨 에이전트와}
- ➕ 신규 제작: {있으면}  ➖ 제외: {안 쓰는 것}
- MCP: {연결할 도구}  · 규제 게이트: {업종}
```
⏸ "이 팀으로 구성할까요? 가감할 것은?"

## STEP 4 · 개선 처방 (운영 문제 → 해결)
STEP 2 진단의 각 문제가 이 팀으로 **어떻게 풀리는지**:
```
🔧 개선 처방
- 과부하 "{직원} 광고점검 주3h" → ad-performance-checker 자동 → {직원}은 전략에 집중
- 갭 "경쟁 모니터링 없음" → competitor-monitor 매일 자동 (사람 0h)
- 책임 공백 "규제 검수 없음" → gate-auditor {업종} 게이트로 발행 전 차단
```
얻는 것을 **사람 시간·리스크·성과**로 명시.

## STEP 5 · 확정 → brand/ 파일 마감
승인되면 채운다:
- `brand/team.md` — 직원 명부 + 사람↔AI 매핑 + 운영(승인권·보고선)
- `brand/org-map.md` §1~4 · `roster.md`(가감 반영) · `marketer.md`(주 담당자)

- 각 파일 `상태: 작성됨 (날짜)`.
