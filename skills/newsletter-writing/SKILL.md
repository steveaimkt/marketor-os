# 뉴스레터 작성 스킬

## 언제 호출하나
- `email-newsletter` 에이전트가 매주
- `content-publisher`가 블로그 채널 콘텐츠를 만들 때
- 사용자가 직접 `claude "Skill newsletter-writing으로 ... 작성"`

## 입력 (parameters)

```yaml
sources:              # 필수
  - title: "..."
    summary: "..."
    url: "..."
  - ... (3~7개 권장)

brand_voice:          # 필수 (없으면 brand-voice 스킬을 먼저 호출)
  tone: "친근하지만 신뢰감 있는"
  vocabulary: ["케어", "챌린지", "가이드", "리뷰"]
  forbidden: ["미백", "주름개선", "의학적"]   # 식약처 광고 가이드 위반어
  emoji_policy: "주제어 옆에 1개씩"

audience:             # 필수
  - persona: "20-30대 여성"
  - core_pain: "모공·피지·새벽 푸석함"

cta:                  # 선택 (기본값: 본 브랜드 상품 안내)
  text: "21일 챌린지 신청하기"
  url: "https://..."
```

## 출력

```yaml
subject_candidates:        # 제목 5종
  - "이번 주, 모공 케어의 새 정답"
  - "왜 다들 토너를 두 번 바르는가"
  - ... (5개)

preheader: "..."          # 미리보기 1줄 (Gmail 인박스에서 보이는 부제)

body_html: |              # 본문 HTML (약 800자)
  <h1>{선택된 제목}</h1>
  <p>{인트로 2문장 · 본주 핵심을 한 문장으로}</p>
  <h2>{소주제 1}</h2>
  <p>{... ~150자}</p>
  ...

word_count: 812
estimated_open_rate: "28~32%"   # 제목·preheader 길이·이모지 기반 추정
```

## 글쓰기 원칙 (Claude는 이걸 항상 지킴)

1. **두괄식** · 첫 문장에 핵심. 마케터 뉴스레터는 오픈 후 평균 7초 동안 결정난다.
2. **숫자·구체** · "더 좋아진" → "21일 후 모공 -38%"
3. **2인칭** · "당신의 모공이..." (3인칭은 거리감)
4. **CTA는 1개만** · 여러 CTA는 클릭률을 떨어뜨림
5. **금기어 자동 제거** · `brand_voice.forbidden`에 있는 단어는 동의어로 자동 대체
6. **이모지** · 주제어 옆 1개, 끝맺음 ❤️ 1개까지

## 제목 작성 5패턴

뉴스레터 제목 5종은 반드시 다음 5패턴을 1개씩 사용:

| 패턴 | 예시 |
|---|---|
| **숫자 호기심** | "왜 다들 토너를 두 번 바르는가" |
| **이번 주 정답** | "이번 주, 모공 케어의 새 정답" |
| **반전·금기** | "스킨케어 7단계는 사실 틀렸다" |
| **데이터 헤드라인** | "구독자 1,247명이 답한 모공 고민 1위" |
| **이름 호출 (TY 변수)** | "{이름}님이 놓치고 있는 것" |

## 검증 (스킬 종료 전 자체 점검)

- [ ] 본문 700~900자
- [ ] 두괄식 (첫 문장이 핵심)
- [ ] 금기어 0개
- [ ] CTA 1개
- [ ] 제목 5종 (5패턴 각 1개)
- [ ] preheader가 제목과 중복되지 않음

## 외부 참조

본 스킬은 다음 사전 자료를 활용할 수 있다 (있을 때만):
- 노션 페이지 "Brand Voice"
- 노션 DB "지난 뉴스레터 성과" (오픈률·클릭률 학습)
