# 톤앤매너 (tone.md)
> 스킬 081(브랜드 스토리·미션 빌더)·082(브랜드 가이드라인 문서화)가 생성·갱신한다. 콘텐츠·카피 계열 스킬(021~040, 043)이 참조.

## 상태
- [x] 작성 완료 · 2026-07-04 (링크드인 검증 포맷 + 트루먼쇼 스타일 기반)

## 1. 보이스 3형용사
1. **단정적** · 결론 먼저, 조건 없이 ("~해야 합니다", "~입니다")
2. **실증적** · 주장 뒤에 항상 실물 (숫자·화면·산출물)
3. **탈기술** · 비개발자가 한 번에 이해하는 언어

## 2. 어휘 사전
| 쓰는 말 | 안 쓰는 말 | 이유 |
|---|---|---|
| 지시문 작성 | 프롬프트 엔지니어링 | 비개발자 언어 원칙 |
| 작동하는 시스템 / 마케팅 팀 | 솔루션, 프레임워크 | 실물 지향 |
| 직원처럼 일하는 에이전트 | AI 어시스턴트 | 자판기=웹앱/직원=에이전트 교육 비유 계승 |
| 저요 | 신청하세요, 문의주세요 | 시그니처 CTA |

## 3. 문체 규칙
- 종결어미: 존댓말 (~입니다/합니다), 권유는 ~하세요
- 이모지: CTA 부근만 (✋👇) · 본문 금지
- 문장 길이: 한 줄 한 개념, 3줄 넘는 문장 금지
- 숫자 표기: 구체 수치 우선 ("많은 분들이" ✗ → "39,357 노출" ✓)

## 4. 채널별 변주
| 채널 | 톤 조정 |
|---|---|
| 링크드인 | 훅 공식 A~G + ❶❷❸ 번호 본문 + "저요" CTA |
| 유튜브 | 훅-시연-데이터-CTA, 구어체 나래이션 |
| 디스코드 | 반말 섞인 친근함 허용, 실시간 감각 |
| 모집 페이지 | 단정적이되 보장 표현 금지 · 실물 증거로 설득 |
