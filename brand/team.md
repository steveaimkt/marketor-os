# 마케팅팀 명부 (team.md) · 사람 + AI 하이브리드

> **이 파일의 정체**: marketing-os는 AI 에이전트만이 아니라 **실제 직원(사람)까지 함께 관리**한다.
> 이 파일이 인력 관리의 정본 — 누가 무슨 일을 하고, 어느 AI 에이전트가 그를 보조하는지, 운영은 어떻게 도는지.
> "마케팅팀 구축하자"(skills/회사셋업)로 채우고, 입·퇴사·역할 변경 때마다 갱신한다.

## 상태
- [ ] 미작성 (템플릿) — "마케팅팀 구축하자"로 채운다.

## 1. 실제 직원 명부 (사람)
| 이름/호칭 | 역할 | 담당 업무 | 부하 | 강점 | 집중할 것(재정의) | AI 위임 |
|---|---|---|---|---|---|---|
| (예: 김OO 팀장) | 총괄 | 전략·예산·승인 | 적정 | 방향 설정 | 전략·의사결정 | 리포트 자동화 |
| | | | 여유/적정/과부하 | | | |

## 2. 사람 ↔ AI 매핑 (누가 어떤 에이전트와 일하나)
> 각 직원이 어떤 AI 에이전트를 "자기 보조"로 쓰는지. 이게 하이브리드 협업의 핵심.

| 직원 | 보조 AI 에이전트 | 무엇을 위임 | 사람이 하는 것(검토·판단) |
|---|---|---|---|
| (예: 콘텐츠 담당) | email-newsletter · content-publisher | 초안 생성·예약 | 최종 검토·발행 승인 |
| | | | |

## 3. 업무 분담표 (사람 / AI / 협업)
| 업무 | 방식 | 담당 |
|---|---|---|
| (예: 주간 뉴스레터) | 🤝 협업 (AI 초안 → 사람 승인) | 콘텐츠담당 + email-newsletter |
| (예: 경쟁사 모니터링) | 🤖 AI 전담 | competitor-monitor |
| (예: 캠페인 전략) | 👤 사람 전담 | 팀장 |

## 4. 운영 관리
| 항목 | 값 |
|---|---|
| 발행·예산 승인권자 | (누구 · 발행/집행 전 이 사람 ⏸ 결재) |
| 보고선 | (AI 산출 → 누구에게 보고) |
| 주간 운영 리듬 | (예: 월 브리핑 · 금 성과 리뷰) |
