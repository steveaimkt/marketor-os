# marketor-os에 오신 것을 환영합니다

당신은 지금 **「마케터를 위한 클로드코드」** 강의의 공식 실습 OS를 설치한 상태입니다.

## 무엇이 설치되었나

| 구성 요소 | 개수 | 용도 |
|---|---|---|
| 에이전트 | 28개 | 리서치·카피·광고·GA4·CRM·확장 자동화 |
| 스킬 | 5개 | 뉴스레터·브랜드톤·광고 카피·6축 품질검수·HTML 리포트 |
| 슬래시 명령 | 4개 | `/marketor-os-welcome`, `/daily-briefing`, `/weekly-report`, `/ax-team-run` |
| 커리큘럼 | Part 01~10 | 강의 대본·슬라이드·실습 자료 |

설치 위치: `~/.claude/plugins/marketor-os/`

## 지금 시작할 것 · Part 01

**현재 v0.1.0 (Part 01만 완성)** · 강의 1주차 입문 자료가 준비되어 있습니다.

다음 파일을 열어 학습을 시작하세요:

```
~/.claude/plugins/marketor-os/curriculum/part01-입문/README.md
```

또는 Claude Code에서:

```
"part01-입문 README 열어줘"
```

Part 01은 7개 클립 (약 70분)으로 구성되어 있으며, 추가 API 키나 MCP 설정 없이 진행할 수 있습니다.

## 앞으로 추가될 내용 (강의 진도와 함께 업데이트)

| 버전 | 추가 콘텐츠 | 예상 시점 |
|---|---|---|
| v0.2 | Part 02 (12 MCP 설정) + `.mcp.json` 활성화 + API 키 setup 스크립트 | 강의 2주차 |
| v0.3 | Part 03 (콘텐츠 파이프라인) + 3개 에이전트 본문 채움 | 3주차 |
| v0.4~ | Part 04~09 점진적 추가 (리서치 5, 카피 5, 광고 6, GA4 3, CRM 3, 확장 3) | 4~9주차 |

```
/plugin update marketor-os
```
