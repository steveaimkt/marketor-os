# /dashboard

"대시보드 열어줘" · "에이전트 대시보드" · "OS 관리 화면" 발화 시 실행.

## 동작

```bash
npm run dashboard   # 포그라운드 단일 인스턴스로 기동 (이미 떠 있으면 EADDRINUSE 안내 후 종료)
open http://localhost:3737
```

이미 실행 중이면 브라우저만 열면 된다. 다른 포트로 띄우려면 `node tools/dashboard/server.mjs --port 3800`.

## 5개 탭

1. **조직도** · 트루먼 → 참모(감사·검수·제작) → 10개 부서(부서장 + 팀원)의 회사형 조직도 + 하네스 동적 팀. agents/ 폴더 구조에서 자동 생성
2. **전체 구조** · 사용자 → 트루먼 → 에이전트 흐름도 + 그룹별 에이전트 + MCP·스킬·커맨드·워크플로 목록
3. **실시간 현황** · 10초마다 갱신. 세션·에이전트 호출·하네스 팀·스킬 호출·자동화·산출물·실패 로그
4. **에이전트 관리** · 정의 파일(.md) 검색 → 편집 → 저장. 저장 전 outputs/status/backups/ 자동 백업 + 외부 변경 충돌 감지
5. **처음이신가요?** · 신규 사용자 온보딩 · 대표 발화 복사 버튼 + 4층 구조 설명 + 안전장치 안내

## 보안

- 127.0.0.1 전용 바인딩 + Host 헤더 화이트리스트(DNS 리바인딩 차단)
- `/api` 전체에 launch별 토큰 헤더 요구(CSRF 차단) · 토큰은 페이지 로드 시 주입
- 편집은 agents/ · skills/ · commands/ 내부 .md만, 심링크 이탈 차단(realpath)

## 관련

- 정적 스냅샷(파일 1장)만 필요하면 `/status` (npm run status)
- 편집 허용 범위는 agents/ · skills/ · commands/ 내부 .md만 (서버가 강제)
- 서버는 127.0.0.1 전용 · 외부 접속 불가
