# /status

"작업 현황 보여줘" · "뭐 돌아가고 있어?" · "현황판 열어줘" 발화 시 실행.

## 동작

```bash
node scripts/status-scan.mjs --open
```

스캔 항목 10가지:
1. 🟢 Claude Code 세션 · 10분 내 갱신 = 작동 중, 24시간 내 = 최근 작업
2. 🤖 에이전트 작동 기록 · 세션 로그에서 서브에이전트 호출 추출 (24시간, 15분 내 = 작동 중)
3. 👥 하네스 팀 (에이전트 팀 모드) · 팀 구성원 + 가동/휴면
4. 🧩 스킬 작동 기록 · 스킬 호출 이력 (24시간)
5. 📋 보유 자산 인벤토리 · 에이전트(파트별)·스킬·커맨드·워크플로 개수
6. ⏰ launchd 자동화 · 등록/가동 상태 + 예약 시각
7. 🔁 워크플로·cron 정의 목록
8. 📦 최근 산출물 (outputs/ 최근 3일)
9. ⚠️ 실패 로그 (logs/failures.jsonl 최근 3건)
10. ✏️ git 작업 중 변경사항

## 라이브 모드

작업을 계속 지켜보고 싶을 때:

```bash
node scripts/status-scan.mjs --watch 30 --open
```

30초마다 재스캔하고 브라우저 페이지가 자동 새로고침된다 (Ctrl+C로 종료).

## 산출물

`outputs/status/작업현황판.html` · 단일 파일, 다크 모드 자동.
