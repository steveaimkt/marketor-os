# Part 2 사전 준비물 — 설치 마라톤 영상 시청 전 체크리스트

> 본 문서는 **Part 2 설치 마라톤 영상을 보기 전에** 학습자가 미리 발급·준비해둘 항목을 정리한 가이드입니다. 사전 준비를 마치면 영상 안의 설치 작업이 약 30~40분 안에 끝납니다 (사전 준비 없이는 약 71분 + 발급 사이트 이동·대기 시간).

## 왜 사전 준비가 필요한가

- 11개 MCP 의 키·토큰 발급은 사이트마다 가입·로그인·OAuth 동의 화면 등 **반복 클릭이 많음**
- 영상 안에서 이걸 다 시연하면 학습자는 따라하다 지치고, 영상은 90분이 넘어감
- **사전 준비 = 자료 모아두기, 영상 시청 = 자동화 작동 확인**으로 역할 분리하면 시간 60% 절감

## 환경 사전 점검 (10분)

영상 시작 전 본인 PC 에 다음이 설치되어 있어야 합니다.

```bash
node --version          # v22 이상
python3 --version       # 3.10 이상
which uv                # uv 패키지 매니저
which ffmpeg            # 영상 트리오 (Step 7) 용
which git               # 깃 클론
which claude            # Claude Code CLI
```

미설치 시:
| 도구 | macOS | Windows (Git Bash / WSL) |
|---|---|---|
| Node 22+ | `brew install node@22` | nodejs.org 에서 LTS 설치 |
| Python 3.10+ | `brew install python@3.11` | python.org 에서 설치 |
| uv | `curl -LsSf https://astral.sh/uv/install.sh \| sh` | 동일 |
| ffmpeg | `brew install ffmpeg` | `choco install ffmpeg` |
| Chrome | 브라우저 다운로드 | 브라우저 다운로드 |
| Claude Code | `npm install -g @anthropic-ai/claude-code` | 동일 |

추가로 marketing-os 깃 리포 클론 (영상 안에서 시연하지만 미리 받아두면 더 빠름):
```bash
git clone <marketing-os 리포 URL> ~/marketing-os
cd ~/marketing-os && cp .env.example .env
```

## MCP 11개 사전 준비물 체크리스트

### Ch 1. 데이터 분석 3개

#### ☐ Step 1 · Google Sheets (10분)
- [ ] [Google Cloud Console](https://console.cloud.google.com) 접속 + 새 프로젝트 `marketing-os` 생성
- [ ] Google Sheets API 활성화
- [ ] OAuth 동의 화면 설정 (User Type 외부, 앱 이름 "Sheets MCP")
- [ ] **테스트 사용자 탭에 본인 이메일 추가** ⚠️ 빠뜨리면 인증 실패
- [ ] OAuth 클라이언트 ID 생성 (**애플리케이션 유형: 데스크톱 앱**) → JSON 다운로드
- [ ] 파일명을 `oauth_credentials.json` 으로 변경

**결과물**: `~/Downloads/oauth_credentials.json` (파일 안에 `"installed"` 키 확인)

#### ☐ Step 2 · GA4 (15분)
- [ ] 같은 GCP 프로젝트에서 Google Analytics Data API 활성화
- [ ] Service Account 생성 (이름 `marketing-os-ga4`)
- [ ] JSON key 다운로드
- [ ] [Google Analytics](https://analytics.google.com) > 관리 > 속성 액세스 관리
- [ ] 서비스 계정 이메일 (`client_email` 값) 을 뷰어 권한으로 추가
- [ ] GA4 속성 ID (9자리 숫자) 메모

**결과물**: `marketing-os-*.json` 파일 + 속성 ID 메모

#### ☐ Step 3 · Firecrawl (2분)
- [ ] [firecrawl.dev](https://firecrawl.dev/app/api-keys) 가입 (Google 로그인 가능)
- [ ] API key 발급 (`fc-xxx` 형식)

**결과물**: API key 메모

### Ch 2. 콘텐츠·영상·디자인 4개

#### ☐ Step 4 · Figma (1분)
- [ ] [Figma](https://figma.com) 무료 계정
- [ ] [Claude.ai](https://claude.ai) 웹사이트 로그인 준비 (영상 안에서 클릭 1번)

**결과물**: Claude.ai 로그인 상태

#### ☐ Step 5 · YouTube Data (5분)
- [ ] 같은 GCP 프로젝트에서 [YouTube Data API v3](https://console.cloud.google.com/apis/library/youtube.googleapis.com) 활성화
- [ ] API key 발급 (`AIza...` 형식)

**결과물**: API key 메모

#### ☐ Step 6 · Higgsfield (1분)
- [ ] Claude.ai 로그인 (Step 4 와 동일)
- [ ] Higgsfield 계정 (Claude.ai 통합 시 자동 생성)

**결과물**: Claude.ai 로그인 상태

#### ☐ Step 7 · 영상 트리오 Hyperframes + HeyGen + ElevenLabs (15분)
- [ ] Hyperframes 글로벌 설치: `npm install -g hyperframes`
- [ ] [HeyGen](https://app.heygen.com) 가입 + Settings > API > "Generate Token"
- [ ] HeyGen API key (`HG_xxx` 형식) 메모
- [ ] [ElevenLabs](https://elevenlabs.io) 가입 + Settings > API Keys > "Create API Key"
- [ ] ElevenLabs API key (`xi_xxx` 형식) 메모

**결과물**: HeyGen key + ElevenLabs key 메모

### Ch 3. 배포와 광고 2개

#### ☐ Step 8 · Buffer (5분)
- [ ] [Buffer](https://buffer.com) 가입 (무료 플랜)
- [ ] LinkedIn / Twitter / Instagram / Facebook 중 **최소 1채널** 연결
- [ ] [Apps 페이지](https://publish.buffer.com/account/apps) > "Create Access Token"
- [ ] 토큰 (`1/xxx` 형식) 메모

**결과물**: Access Token 메모

#### ☐ Step 9 · Meta + Google Ads (6분 발급 + **1~2일 승인 대기**)

**Meta Ads** (영상 안에서 OAuth 1클릭):
- [ ] Facebook 광고 계정 보유 (활성 캠페인 0개여도 OK)
- [ ] [Claude.ai](https://claude.ai) 로그인 상태

**Google Ads** ⚠️ **영상 촬영 1~2일 전에 신청 필수**:
- [ ] [Google Ads Manager Account](https://ads.google.com) 보유
- [ ] [API Center](https://ads.google.com/aw/apicenter) > "Apply for Basic access" 신청
- [ ] 회사 정보 + 사용 목적 + 예상 호출량 입력 → 제출
- [ ] **1~2일 후 이메일로 Developer Token 수신**
- [ ] 토큰 (22자 문자열) 메모

**결과물**: Developer Token 메모 + ADC 인증 (`gcloud auth application-default login` 1회)

### Ch 4. 협업과 관리 2개

#### ☐ Step 10 · Notion (1분)
- [ ] Notion 워크스페이스 보유
- [ ] Claude.ai 로그인 (Step 4 와 동일)

**결과물**: Claude.ai 로그인 상태

#### ☐ Step 11 · Discord (10분)
- [ ] 본인 Discord 서버 보유 (또는 봇 추가 권한 있는 서버)
- [ ] [Discord Developer Portal](https://discord.com/developers/applications) > "New Application" > 이름 `marketing-os`
- [ ] 좌측 "Bot" > "Reset Token" > 토큰 (`MTxxx` 형식) 메모 (**한 번만 표시됨**)
- [ ] Privileged Gateway Intents 3개 모두 ON (PRESENCE / SERVER MEMBERS / MESSAGE CONTENT)
- [ ] OAuth2 URL Generator > Scopes: bot, Permissions: Send Messages·Embed Links·Read Messages
- [ ] 생성된 URL 로 본인 서버에 봇 초대
- [ ] Discord 설정 > 고급 > 개발자 모드 활성화
- [ ] 메시지 발송할 채널 우클릭 > "ID 복사" → 메모

**결과물**: Bot Token + Channel ID 메모

## 사전 준비 완료 체크 — 영상 시작 직전 점검

영상 시작 직전 다음을 책상 옆에 정리해두세요. **메모장 1개에 다 적어두면 영상 안에서 붙여넣기 한 번에 끝납니다.**

```
[환경]
  ☐ Node 22+ · Python 3.10+ · uv · ffmpeg · Chrome · Claude Code
  ☐ marketing-os 깃 클론 완료 + .env 파일 생성됨
  ☐ Claude.ai 웹사이트 로그인 상태

[파일 2개 (Downloads 폴더 또는 메모해둔 경로)]
  ☐ oauth_credentials.json    (Step 1 Google Sheets)
  ☐ marketing-os-*.json       (Step 2 GA4 Service Account)

[메모장에 적어둘 키 8개]
  ☐ GA4 속성 ID            (9자리 숫자)
  ☐ FIRECRAWL_API_KEY      (fc-xxx)
  ☐ YOUTUBE_API_KEY        (AIza...)
  ☐ HEYGEN_API_KEY         (HG_xxx)
  ☐ ELEVENLABS_API_KEY     (xi_xxx)
  ☐ BUFFER_ACCESS_TOKEN    (1/xxx)
  ☐ GOOGLE_ADS_DEVELOPER_TOKEN (22자)
  ☐ DISCORD_BOT_TOKEN      (MTxxx)
  ☐ DISCORD_CHANNEL_ID     (숫자)

[Claude.ai 통합 3개 — 영상 안에서 클릭 1번씩]
  ☐ Figma · Higgsfield · Notion
```

## 사전 준비 시간 합계

| 패턴 | 사전 준비 시간 |
|---|---|
| 환경 설치 (Node·Python·uv·ffmpeg·Chrome·Claude Code) | 10분 |
| GCP 1회 프로젝트 + API 3종 활성화 (sheets·ga4·youtube) | 15분 |
| Service Account + JSON 2개 (sheets·ga4) | 15분 |
| API key 5종 (firecrawl·youtube·heygen·elevenlabs·buffer) | 15분 |
| Discord Bot Token + 서버 초대 | 10분 |
| Google Ads Developer Token **신청만** (승인은 별도 대기) | 5분 |
| Claude.ai 로그인 + 계정 3종 (figma·higgsfield·notion 사용자만) | 5분 |
| **합계 (Google Ads 승인 대기 별도)** | **약 75분** |

영상 시작 전에 위 작업을 마치면 **영상 안의 설치 작업은 약 30~40분**에 끝납니다.

## Google Ads 만 예외

Google Ads Developer Token 은 신청 후 1~2일 승인 대기가 필요합니다.

→ **영상 촬영하기 최소 2일 전에 신청 완료**해두는 것을 강력 권장합니다. 영상 안에서는 토큰을 받아둔 상태로 시연.

만약 신청만 영상 안에서 시연하고 싶다면, Step 9.2 신청 절차만 보여주고 "1~2일 후 토큰 받으면 `MCP 전체 설치하자` 재호출" 안내로 마무리합니다.
