# 클립 1-1 실습 · MCP가 켜진 상태 직접 확인하기

> 이 클립은 개념 클립이라 설치는 없습니다.
> 대신 **이미 켜져 있는 MCP를 눈으로 확인**하고, 다음 11개 클립에서 무엇이 추가될지 미리 봅니다.

## 목표 산출물

- `claude mcp list` 출력 캡처 1장
- 본인 환경에 이미 들어 있는 MCP 목록 파악
- 12개 중 몇 개가 미리 켜져 있는지 카운트

## 단계별 명령

```bash
# 1) marketing-os 폴더로 이동
cd marketing-os

# 2) 현재 등록된 MCP 목록 확인
claude mcp list
```

처음이라면 다음 둘 중 하나의 결과가 나옵니다.

**(A) 비어 있음**
```
No MCP servers configured.
```
→ 정상. 다음 클립부터 한 개씩 추가하면서 늘어납니다.

**(B) 이미 일부 켜져 있음**
```
google-drive    ✅ ready    (Claude.ai integration)
gmail           ✅ ready    (Claude.ai integration)
calendar        ✅ ready    (Claude.ai integration)
notion          ✅ ready    (Claude.ai integration)
```
→ Claude.ai 계정에 이미 연동해 둔 통합이 자동 노출된 것. 이 강의에서 별도로 설치할 필요가 없습니다.

## 사고 실험 · "지금 내 마케팅 업무 중 MCP가 가장 필요한 곳은?"

다음 표를 본인 업무로 채워 보세요. 채우는 동안 12개 MCP 중 어떤 것이 본인에게 가장 먼저 필요한지 보입니다.

| 매주 반복 작업 | 소요 시간 | 어떤 도구를 만지나 | 필요한 MCP 후보 |
|---|---|---|---|
| 예) 광고 성과를 시트에 옮기기 | 60분 | Meta Ads + Google Sheets | `meta-ads`, `google-sheets` |
| | | | |
| | | | |
| | | | |

**판정 기준**: 같은 도구 이름이 2회 이상 등장하면, 그 MCP는 본인에게 우선순위 상위입니다.

## .mcp.json 미리 보기

다음 클립부터 채워 갈 `.mcp.json`은 최종적으로 다음 모양이 됩니다 (지금은 비어 있어도 정상).

```json
{
  "mcpServers": {
    "google-sheets":   { "command": "npx", "args": ["..."] },
    "ga4":             { "command": "npx", "args": ["..."] },
    "firecrawl":       { "command": "npx", "args": ["..."] },
    "figma":           { "command": "npx", "args": ["..."] },
    "youtube-data":    { "command": "npx", "args": ["..."] },
    "buffer":          { "command": "npx", "args": ["..."] },
    "meta-ads":        { "command": "npx", "args": ["..."] },
    "google-ads":      { "command": "npx", "args": ["..."] },
    "discord":         { "command": "npx", "args": ["..."] }
  }
}
```

Higgsfield · Notion · Gmail · Google Drive는 Claude.ai 통합 방식이라 `.mcp.json`에 별도 등록하지 않습니다.

## 트러블슈팅

| 증상 | 원인 | 해결 |
|---|---|---|
| `claude: command not found` | Part 1에서 Claude Code 설치 누락 | Part 1의 [2-1 클립](../../part01-입문/대본/2-1-Claude-Code-설치.md) 다시 실행 |
| `mcp` 서브명령이 없다고 나옴 | Claude Code 구버전 | `npm i -g @anthropic-ai/claude-code` 로 최신화 |
| Claude.ai 통합이 안 보임 | Claude 계정에 통합을 추가 안 함 | Claude.ai 웹 → Settings → Integrations 에서 활성화 |
