자주 묻는 오해
MCP에 대한 3가지 오해
Q1
MCP는 챗봇 도구 사용(Tool Use)과 같은 것?
아니오 · 층이 다름.
Tool Use는 AI가 도구를 "어떻게 호출하는가". MCP는 그 도구가 "어디에 어떤 모양으로 정의되어 있는가". 한 세트로 쓰임.
Q2
그냥 API를 직접 호출하면 안 되나?
가능 · 다만 비효율.
API마다 스키마 · 인증 · 에러 처리를 직접 코딩해야 함. MCP는 "한 번 설치 = 평생 재사용" 규격으로 표준화.
Q3
MCP 서버는 누가 만드나? 안전한가?
서비스 회사 · Anthropic · 커뮤니티.
본 강의는 공식 · 검수된 것만 사용. 정체 불명 서버는 절대 사용 ✕ (프롬프트 주입 위험).
